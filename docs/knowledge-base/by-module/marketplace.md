# Маркетплейс

---

## Что это и зачем

**Маркетплейс** — это магазин плагинов (расширений) для системы ПРИВОД. Здесь можно установить дополнительные модули, которые расширяют функциональность системы.

Базовая система покрывает большинство задач строительной компании. Но если вам нужно что-то специфическое — ищите в маркетплейсе:
- Расширенные шаблоны документов
- Отраслевые калькуляторы
- Интеграции с нишевыми системами
- Специализированные отчёты

## Часть 3: Работа с плагинами

### Установка

1. Откройте каталог маркетплейса
2. Найдите нужный плагин (поиск или по категории)
3. Нажмите **«Установить»**
4. Подтвердите (для платных — оплатите)
5. Плагин активируется автоматически

### Настройка

Некоторые плагины требуют настройки после установки:
- API-ключи (для интеграций)
- Параметры (для калькуляторов)
- Шаблоны (для генераторов документов)

Настройки плагина доступны на его карточке.

### Отзывы

Каждый плагин имеет:
- **Рейтинг** — от 1 до 5 звёзд
- **Отзывы** — комментарии пользователей
- **Количество установок** — сколько компаний используют

### Удаление

Если плагин больше не нужен:
1. Откройте карточку плагина
2. Нажмите **«Удалить»**
3. Подтвердите — плагин деактивируется

> Данные, созданные плагином, сохраняются. Если установите снова — данные будут на месте.

## Типичный сценарий

### Установка плагина «Калькулятор бетона»

```
1. Прораб часто считает объём бетона вручную
   (длина × ширина × высота, с учётом армирования)

2. Находит в маркетплейсе:
   → Категория: Производство
   → «Калькулятор бетонных работ»
   → Рейтинг: 4.7 ⭐ (128 установок)
   → Цена: бесплатно

3. Нажимает «Установить» → готово ✅

4. Теперь при создании работы «Бетонирование»:
   → Появляется кнопка «Рассчитать объём»
   → Вводит размеры: 12 × 8 × 0.3 м
   → Марка: М300, армирование: 120 кг/м³
   → Калькулятор: 28.8 м³ бетона, 3 456 кг арматуры
   → Данные подставляются в заявку на материалы
```

## Частые вопросы

**Кто может устанавливать плагины?**
Только пользователи с ролью **Администратор** (ADMIN).

**Безопасны ли плагины?**
Все плагины в маркетплейсе проходят проверку. Они работают в изолированной среде и не имеют доступа к вашим данным, кроме тех, что нужны для их работы.

**Можно ли создать свой плагин?**
Да, для плана ENTERPRISE. Обратитесь к службе поддержки для получения документации по разработке плагинов.

**Что будет, если удалить плагин?**
Плагин деактивируется, но данные, которые он создал, останутся. При повторной установке — данные будут доступны.

**Как оплачиваются платные плагины?**
Так же, как подписка — через ЮKassa. Стоимость плагинов добавляется к ежемесячному счёту.
