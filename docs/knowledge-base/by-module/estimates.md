# Сметы (ЛСР)

---

## Что это и зачем

Смета — это документ, в котором посчитана стоимость работ и материалов по нормативам. Каждая строка — расценка (например, «Кладка кирпича — 1 500 руб/м3 по ФЕР»), умноженная на объём (100 м3 = 150 000 руб). В конце — итого с накладными расходами и сметной прибылью.

Без сметы не согласуют ни один строительный проект. Это «паспорт стоимости» ваших работ. Из сметы система узнает объёмы и цены, рассчитает бюджет и покажет маржу.

---

## Где найти

**Путь:** Главное меню → **Сметы**

---

## Список смет

**Что видно в таблице:**

| Колонка | Что показывает |
|---------|---------------|
| Название | Имя сметы + привязанная спецификация мелким шрифтом |
| Проект | К какому объекту относится |
| Спецификация | Из какой спецификации создана |
| Статус | Цветная метка: черновик, в работе, активная |
| План | Плановая стоимость (итого по смете) |
| Факт | Сколько уже потрачено |
| Отклонение % | Зелёным — экономия, красным — перерасход |

**Вкладки-фильтры:**
- **Все** — все сметы
- **Черновик** — ещё в работе, не утверждены
- **В работе** — согласовываются
- **Активная** — утверждённые, действующие

**Кнопки:**
- **Импорт ЛСР** — загрузить смету из ГРАНД-Сметы (xlsx-файл)
- **Создать** — создать смету вручную

## Как создать смету

### Способ 1: Импорт из ГРАНД-Сметы (рекомендуется)

Если у вас есть готовая смета в программе ГРАНД-Смета (файл .xlsx) — импортируйте её. Это самый быстрый способ.

**Пошагово:**

1. Нажмите кнопку **«Импорт ЛСР»** на странице списка смет (или перейдите в Сметы → Импорт ЛСР)

2. **Шаг 1 — Загрузка файла:**
   - Перетащите xlsx-файл сметы в окно загрузки (или нажмите «Выбрать файл»)
   - Выберите проект, к которому относится смета
   - Система распознает формат ГРАНД-Сметы автоматически

3. **Шаг 2 — Предпросмотр:**
   - Система покажет распознанные данные: разделы, позиции, объёмы, расценки
   - Проверьте, что разделы и позиции определились правильно
   - Сравните итоговые суммы с оригиналом сметы

4. **Шаг 3 — Импорт:**
   - Нажмите **«Импортировать»**
   - Система создаст смету со всей иерархией: разделы → позиции → ресурсы

**Что распознаёт парсер:**
- Разделы сметы (заголовки)
- Позиции по кодам ГЭСН, ФЕР, ТЕР, ФСБЦ, ТЦ (и транспортные расценки)
- Позиции оборудования (код с суффиксом «О»)
- Объёмы, расценки, итоги по каждой позиции
- Накладные расходы и сметную прибыль
- Итого по разделам и по смете в целом

> **Подсказка:** Система поддерживает формат ГРАНД-Сметы «Методика 2020 РИМ» (ресурсно-индексный метод). Если файл в другом формате — попробуйте экспортировать из ГРАНД-Сметы в xlsx повторно.

### Способ 2: Создать из спецификации

Если у вас есть загруженная спецификация материалов:

1. Нажмите **«Создать»** на странице списка смет
2. Заполните форму:
   - **Название** (обязательно) — например, «ЛСР — Общестроительные работы»
   - **Проект** (обязательно) — выберите проект
   - **Спецификация** (обязательно) — выберите спецификацию этого проекта (система подтянет позиции)
   - **Договор** — привяжите к договору (необязательно)
   - **Примечания** — свободный текст
3. Нажмите **«Создать смету»**

### Способ 3: Создать вручную

Перейдите на страницу сметы и добавляйте позиции по одной: название работы, единица, количество, расценка. Подходит для небольших смет или дополнительных работ.

## Структура сметы

Смета имеет иерархическую структуру:

```
Локальная смета (ЛСР)
├── Раздел 1: Земляные работы
│   ├── Позиция 1: Разработка грунта (ГЭСН 01-01-003)
│   │   ├── Рабочие: машинист, землекоп
│   │   ├── Материалы: —
│   │   └── Механизмы: экскаватор, самосвал
│   ├── Позиция 2: Обратная засыпка (ГЭСН 01-02-001)
│   └── ...
├── Раздел 2: Фундаменты
│   ├── Позиция 3: Монтаж опалубки (ФЕР 06-01-001)
│   ├── Позиция 4: Бетонирование (ФЕР 06-01-002)
│   └── ...
└── ...
```

**Типы строк:**
- **Раздел** — заголовок группы работ (например, «Земляные работы»)
- **Позиция** — конкретная расценка с кодом (ГЭСН/ФЕР/ТЕР), единицей и объёмом
- **Ресурс** — составляющая позиции: рабочая сила, материалы или механизмы

## Страница сметы (детальный вид)

Нажмите на смету в списке — откроется детальная страница.

**Два вида отображения** (переключатель вверху):
- **ЛСР (дерево)** — иерархический вид: разделы → позиции → ресурсы. Показывает смету так, как она выглядит в ГРАНД-Смете
- **Таблица** — плоский список всех позиций для быстрого редактирования

### Показатели (карточки вверху)

**Для нормативной сметы (импортированной из ГРАНД-Сметы):**

| Показатель | Что значит |
|-----------|-----------|
| ПЗ (прямые затраты) | Стоимость работ и материалов «как есть» — руки + материалы + механизмы |
| НР (накладные расходы) | Содержание компании: зарплата управленцев, аренда, охрана (% от ФОТ) |
| СП (сметная прибыль) | Нормативная прибыль подрядчика (% от ФОТ) |
| Итого с НДС | Полная стоимость с налогом 20% |

**Для обычной сметы:**

| Показатель | Что значит |
|-----------|-----------|
| План | Плановая стоимость по смете |
| Заказано | Сколько уже заказано у поставщиков |
| Оплачено | Сколько фактически выплачено |
| Остаток | Разница: план минус оплачено |

### Таблица позиций (плоский вид)

| Колонка | Описание |
|---------|----------|
| Наименование | Название работы или материала |
| Количество | Объём в единицах измерения (м3, м2, шт, т) |
| Цена за единицу | Стоимость одной единицы. **Можно редактировать** — нажмите на ячейку |
| Сумма (план) | Количество x Цена = плановая стоимость |
| Заказано | Сколько заказано |
| Оплачено | Сколько оплачено |
| Остаток | Разница (зелёный — экономия, красный — перерасход) |
| Исполнение | Полоска прогресса: сколько % от плана освоено |

> **Подсказка:** Цену за единицу можно редактировать прямо в таблице — кликните на ячейку, введите новое значение, нажмите Enter. Система автоматически пересчитает суммы.

## Итоги сметы

В нижней части сметы показывается финансовая сводка:

| Строка | Что значит |
|--------|-----------|
| **Прямые затраты (ПЗ)** | Сумма стоимости работ, материалов и механизмов |
| **Накладные расходы (НР)** | Обычно 80-120% от фонда оплаты труда (ФОТ). Содержание компании |
| **Сметная прибыль (СП)** | Обычно 50-65% от ФОТ. Нормативная прибыль подрядчика |
| **Итого без НДС** | ПЗ + НР + СП |
| **НДС 20%** | Налог на добавленную стоимость |
| **Итого с НДС** | Финальная стоимость сметы |

> **Пример:** Прямые затраты 10 000 000 руб., ФОТ в них 3 000 000 руб. Накладные 100% от ФОТ = 3 000 000. Прибыль 60% от ФОТ = 1 800 000. Итого без НДС = 14 800 000. НДС 20% = 2 960 000. **Итого с НДС = 17 760 000 руб.**

## Стадии сметы

| Стадия | Цвет | Что означает |
|--------|------|-------------|
| **Черновик** | Серый | Смета в разработке. Можно редактировать, добавлять/удалять позиции |
| **В работе** | Синий | Смета на согласовании или проверке |
| **Активная** | Зелёный | Смета утверждена. Используется для бюджета и финансовой модели |

Для нормативной сметы (ЛСР) дополнительно есть кнопка **«Рассчитать»** — пересчитывает итоги по всем разделам.

---

## Как редактировать смету

1. Откройте смету (нажмите на неё в списке)
2. В режиме **«Таблица»** — кликните на ячейку цены для редактирования
3. Для изменения общих данных (название, договор, примечания) — нажмите **«Редактировать»** на странице сметы
4. Изменения сохраняются автоматически при нажатии Enter или при переходе на другую ячейку

## Дополнительные возможности

### Индексы Минстроя

**Путь:** Сметы → Индексы Минстроя

Индексы — это коэффициенты пересчёта из базовых цен (2001 года) в текущие. Например, индекс 9.5 означает: базовые 100 руб. = текущие 950 руб. Обновляются Минстроем ежеквартально.

- Загрузите файл индексов
- Примените к смете — система пересчитает все расценки в текущие цены

### Калькулятор объёмов

Помогает рассчитать объём работ по формулам (площадь стен, объём бетона, погонные метры).

### Сравнение смет

Позволяет сравнить две версии сметы: что изменилось, где увеличилась стоимость, где уменьшилась.

### Сверка с ФМ

Сравнение данных сметы с финансовой моделью — проверка, совпадают ли цифры.

### OCR-сканер

Распознавание сметы с фотографии или скана (PDF). Система извлечёт позиции и загрузит в электронном виде.

## Экспорт и печать

На странице сметы есть кнопка **«Печать»** (иконка принтера).

**Печатная форма включает:**
- Заголовок: название сметы, проект, дата
- Таблица позиций: номер, наименование, единица, количество, расценка, сумма
- Итоги по разделам
- Накладные расходы и сметная прибыль
- Итого с НДС
- Реквизиты организации

Также доступен экспорт сметы в файл для обмена с другими программами.

## Связь с другими модулями

| Модуль | Как связан |
|--------|-----------|
| **Проект** | Смета привязывается к проекту |
| **Спецификация** | Смета создаётся на основе спецификации (позиции подтягиваются) |
| **Финансовая модель (ФМ)** | Сметные цены передаются в ФМ как «сметная стоимость» (estimatePrice) |
| **КП** | Коммерческое предложение формируется из ФМ, куда попали данные сметы |
| **Договор** | Смета может быть привязана к договору (приложение) |
| **КС-2** | Акт выполненных работ ссылается на позиции сметы |
| **База расценок** | Нормативные расценки (ФЕР/ТЕР/ГЭСН) используются при составлении сметы |

**Цепочка данных:**
```
Проектная документация → Смета (ЛСР) → ФМ → КП → Договор → КС-2
```

## Частые вопросы

**Какие форматы файлов поддерживает импорт?**
XLSX-файлы из ГРАНД-Сметы (Методика 2020 РИМ — ресурсно-индексный метод). Поддерживаются расценки ГЭСН, ФЕР, ТЕР, ФСБЦ, ТЦ и транспортные коды.

**Суммы после импорта не совпадают с оригиналом — что делать?**
Проверьте, что при экспорте из ГРАНД-Сметы были включены все разделы. Также убедитесь, что в файле есть итоговые строки «Итого по разделу» — парсер использует их для проверки.

**Можно ли редактировать импортированную смету?**
Да. В режиме «Таблица» можно изменить цену за единицу — кликните на ячейку. Для нормативной сметы (в режиме «ЛСР») рекомендуется вносить изменения в ГРАНД-Смете и импортировать повторно.

**Что такое ПЗ, НР, СП?**
ПЗ — прямые затраты (руки + материалы + механизмы). НР — накладные расходы (содержание компании, % от ФОТ). СП — сметная прибыль (нормативная прибыль, % от ФОТ). Подробнее — в [Глоссарии](../01-concepts.md).

**Как создать новую версию сметы?**
На странице сметы нажмите кнопку **«Создать версию»**. Создастся копия сметы, которую можно редактировать без влияния на оригинал. Удобно для отслеживания изменений.

**Как передать данные сметы в финансовую модель?**
При импорте ЛСР данные автоматически передаются в ФМ проекта (если ФМ уже создана). Также можно вручную: на странице ФМ есть кнопка «Импорт ЛСР».

**Что такое индекс Минстроя?**
Число, на которое умножают стоимость в базовых ценах (2001 года), чтобы получить текущую стоимость. Обновляется ежеквартально. Без индекса ваша смета будет в ценах 20-летней давности.
