# Юридический модуль

---

## Что это и зачем

**Юридический модуль** — управление судебными делами, претензиями и шаблонами юридических документов. Строительный бизнес — один из самых «судебных»: споры с заказчиками, подрядчиками, надзорными органами.

---

## Где найти

**Путь:** Главное меню → **Юридический отдел**

| Страница | URL | Что делает |
|----------|-----|-----------|
| **Дела** | /legal/cases | Реестр судебных дел и претензий |
| **Шаблоны** | /legal/templates | Шаблоны юридических документов |

---

## Часть 1: Реестр дел

| Поле | Описание |
|------|----------|
| **Номер дела** | Внутренний номер |
| **Тип** | Претензия / Досудебное / Арбитраж / Суд общей юрисдикции |
| **Контрагент** | С кем спор |
| **Проект** | По какому проекту |
| **Сумма требований** | Сколько требуем / требуют от нас |
| **Наша роль** | Истец / Ответчик |
| **Статус** | Новое → В работе → На рассмотрении → Завершено (выиграно/проиграно/мировое) |
| **Юрист** | Кто ведёт |
| **Следующая дата** | Ближайшее судебное заседание / срок ответа |

### Типичные причины споров в строительстве

| Причина | Пример |
|---------|--------|
| **Задержка оплаты** | Заказчик не оплатил КС-2 в срок |
| **Некачественные работы** | Подрядчик сделал брак, отказывается исправлять |
| **Нарушение сроков** | Подрядчик не уложился в срок |
| **Изменение объёмов** | Спор о доп. работах (были или не были согласованы) |
| **Гарантийные претензии** | Дефект обнаружен в гарантийный период |
| **Штрафы надзорных органов** | Нарушение ОТ, экология, разрешения |

---

## Часть 2: Шаблоны документов

Готовые шаблоны юридических документов:

| Шаблон | Назначение |
|--------|-----------|
| **Претензия** | Досудебная претензия контрагенту |
| **Ответ на претензию** | Ответ на полученную претензию |
| **Исковое заявление** | Подача в арбитражный суд |
| **Мировое соглашение** | Урегулирование спора без суда |
| **Доверенность** | На представление интересов в суде |
| **Гарантийное письмо** | Обязательство выполнить/оплатить |
