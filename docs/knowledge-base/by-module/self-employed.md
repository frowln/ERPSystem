# Самозанятые

---

## Что это и зачем

**Самозанятые** — модуль для работы с подрядчиками, зарегистрированными как самозанятые (плательщики НПД — налога на профессиональный доход). В строительстве часто привлекают самозанятых: отделочники, электрики, сварщики.

Зачем отдельный модуль:
- Самозанятые — не штатные сотрудники, у них другой документооборот
- Нужно проверять статус самозанятого (действующий ли)
- Оплата идёт по актам, а не по табелю
- Для компании нет НДФЛ и взносов (экономия 30%+)

## Часть 1: Реестр самозанятых

| Поле | Описание |
|------|----------|
| **ФИО** | Полное имя |
| **ИНН** | Для проверки статуса НПД |
| **Телефон / Email** | Контакты |
| **Специализация** | Электрик, сварщик, отделочник и т.д. |
| **Статус НПД** | Действующий / Не проверен / Снят с учёта |
| **Проект** | На каком объекте работает |
| **Договор** | Номер договора ГПХ |
| **Рейтинг** | Оценка по результатам работы (1–5) |

> **Важно:** Перед каждой выплатой система проверяет статус самозанятого через ФНС. Если статус снят — платить нельзя (иначе это будет зарплата с НДФЛ и взносами).

---

## Часть 2: Оплаты

| Поле | Описание |
|------|----------|
| **Самозанятый** | Кому платим |
| **Акт** | Акт выполненных работ (основание) |
| **Сумма** | Без НДФЛ и взносов |
| **Чек** | Самозанятый формирует чек в приложении «Мой налог» |
| **Статус** | Черновик → На оплату → Оплачено |

### Процесс оплаты

```
1. Самозанятый выполнил работу
2. Подписан акт выполненных работ
3. Самозанятый формирует чек в «Мой налог»
4. Бухгалтер проверяет чек и оплачивает
5. Расход попадает в бюджет проекта
```

## Частые вопросы

**Чем самозанятый отличается от штатного?**
Самозанятый — не в штате, работает по договору ГПХ. Компания не платит НДФЛ и взносы. Самозанятый сам платит 6% НПД.

**Можно ли привлекать самозанятых на стройку?**
Да, но есть ограничения. Самозанятый не может быть бывшим сотрудником (менее 2 лет с увольнения). Договор должен быть на конкретный результат, а не на процесс.

**Что если у самозанятого снят статус НПД?**
Система предупредит. Платить нельзя — иначе ФНС переквалифицирует это в трудовые отношения с доначислением НДФЛ и взносов.
