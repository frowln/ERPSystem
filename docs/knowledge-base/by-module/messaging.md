# Мессенджер

---

## Что это и зачем

**Мессенджер** — это встроенный чат для общения внутри компании. Как Telegram или WhatsApp, но внутри системы ПРИВОД — и всё привязано к рабочим проектам.

Зачем нужен отдельный мессенджер:
- Рабочая переписка не теряется в личных мессенджерах
- Каналы привязаны к проектам — вся информация в одном месте
- Файлы, фото, голосовые сообщения — прямо в чате
- Можно закрепить важное сообщение, добавить в избранное
- Треды (ветки обсуждения) — не засоряют общий чат
- Видео- и аудиозвонки — без внешних сервисов

---

## Где найти

**Путь:** Главное меню → **Мессенджер**

| Страница | URL | Что делает |
|----------|-----|-----------|
| **Чат** | /messaging | Основной интерфейс: каналы + сообщения |
| **Избранное** | /messaging/favorites | Сохранённые сообщения с заметками |
| **Звонки** | /messaging/calls | История звонков, активные звонки |

## Часть 1: Каналы

**Канал** — это место для общения группы людей. Как «чат» в Telegram.

### Типы каналов

| Тип | Описание | Кто видит | Пример |
|-----|----------|-----------|--------|
| **Публичный** | Открытый для всех сотрудников | Все в организации | «Общий», «Новости компании» |
| **Приватный** | Только для приглашённых | Только участники | «Проект ЖК Южный — команда» |
| **Личные сообщения** | Переписка один на один | Только 2 человека | Директор ↔ Главный инженер |

### Как создать канал

1. Нажмите **«Создать канал»**
2. Заполните:

| Поле | Описание |
|------|----------|
| **Название** | Например: «объект-южный-парк» |
| **Описание** | О чём этот канал |
| **Тип** | Публичный / Приватный |
| **Участники** | Кого добавить (для приватного) |

3. Канал создан — можно писать

### Роли участников

| Роль | Что может |
|------|-----------|
| **Владелец** (OWNER) | Всё: удалить канал, управлять участниками |
| **Администратор** (ADMIN) | Добавлять/удалять участников, закреплять сообщения |
| **Участник** (MEMBER) | Читать и писать сообщения |

---

## Часть 2: Сообщения

### Типы сообщений

| Тип | Описание |
|-----|----------|
| **Текст** | Обычное текстовое сообщение |
| **Файл** | Прикреплённый документ (PDF, XLSX, DOCX...) |
| **Изображение** | Фотография (открывается в просмотрщике) |
| **Голосовое** | Голосовое сообщение (запись с микрофона) |
| **Системное** | Автоматическое уведомление (кто-то вступил в канал и т.д.) |

### Что можно сделать с сообщением

| Действие | Описание |
|----------|----------|
| **Ответить** | Цитата + ваш ответ (в общем чате) |
| **Ответить в треде** | Открывается боковая ветка обсуждения |
| **Реакция** | Поставить эмодзи (👍 ❤️ 😂 🔥 и т.д.) |
| **Закрепить** | Важное сообщение видно наверху канала |
| **В избранное** | Сохранить с заметкой для себя |
| **Переслать** | Отправить в другой канал |
| **Редактировать** | Изменить текст (пометка «(ред.)») |
| **Удалить** | Удалить для всех (необратимо) |

---

## Часть 3: Треды (ветки обсуждения)

**Тред** — это обсуждение, привязанное к конкретному сообщению. Открывается в боковой панели и не засоряет основной чат.

Зачем нужны:
- Кто-то написал: «Нужно обсудить фундамент блока Б»
- 5 человек начинают обсуждать → 20 сообщений
- Без тредов — основной чат «утонет» в обсуждении
- С тредами — обсуждение в отдельной ветке, основной чат чистый

На карточке сообщения с тредом видно:
- Количество ответов (например: «5 ответов»)
- Дата последнего ответа

---

## Часть 4: Избранное

**Путь:** Мессенджер → **Избранное**

Любое сообщение можно добавить в избранное — чтобы не потерять важную информацию.

При добавлении можно написать **заметку** — зачем вы это сохранили:
- Сообщение: «Арматура 12 мм — 45 тонн, привезут в среду»
- Ваша заметка: «Проверить при приёмке на складе»

В избранном видно: текст сообщения, канал, дата, ваша заметка.

---

## Часть 5: Статусы пользователей

Каждый пользователь имеет **статус доступности**:

| Статус | Значок | Описание |
|--------|--------|----------|
| **В сети** (ONLINE) | 🟢 | Активен в системе |
| **Отошёл** (AWAY) | 🟡 | Нет активности некоторое время |
| **Занят** (BUSY) | 🔴 | Не беспокоить по мелочам |
| **Не беспокоить** (DO_NOT_DISTURB) | ⛔ | Уведомления отключены |
| **Не в сети** (OFFLINE) | ⚪ | Вышел из системы |

Можно также задать **текстовый статус** (например: «На объекте до 15:00») и **эмодзи статуса**.

### Индикаторы

- **«Печатает...»** — видно, когда кто-то набирает сообщение
- **Доставлено / Прочитано** — статус доставки сообщения
- **Счётчик непрочитанных** — на каждом канале видно, сколько новых сообщений

## Часть 6: Звонки

**Путь:** Мессенджер → **Звонки**

Можно совершать **аудио-** и **видеозвонки** прямо из мессенджера.

### Типы звонков

| Тип | Описание | Возможности |
|-----|----------|-------------|
| **Аудиозвонок** | Только голос | Микрофон вкл/выкл |
| **Видеозвонок** | Голос + видео | Камера, микрофон, демонстрация экрана |

### Статусы звонка

| Статус | Описание |
|--------|----------|
| **Ожидание** | Звонок идёт, ждём ответа |
| **Активный** | Разговор идёт |
| **Завершён** | Звонок окончен нормально |
| **Отменён** | Звонящий отменил до ответа |
| **Пропущен** | Никто не ответил |

### Возможности во время звонка

- Включить/выключить **микрофон**
- Включить/выключить **камеру** (для видеозвонков)
- **Демонстрация экрана** — показать свой экран другим участникам
- **Ссылка-приглашение** — отправить гостю ссылку для подключения без аккаунта

### Статистика звонков

На странице звонков видно:
- Активные звонки (сейчас)
- Всего звонков
- Общее время в звонках (минут)
- История: кто, когда, сколько длился

---

## Часть 7: Поиск

В мессенджере есть **поиск**:
- По тексту сообщений в канале
- По названию канала
- По имени пользователя

Удобно для поиска старых обсуждений: «Что решили по арматуре для блока Б?»

## Типичные сценарии

### Сценарий 1: Оперативное решение на объекте

```
1. Прораб пишет в канал «Объект — ЖК Южный парк»:
   → «Прибыл бетон, но марка М250 вместо М300.
      Что делать? Фото накладной:» [фото]

2. Главный инженер отвечает в треде:
   → «Категорически нельзя использовать.
      Отправляйте обратно, заказывайте M300»

3. Снабженец в том же треде:
   → «Позвоню поставщику. Замена будет через 3 часа»

4. Прораб закрепляет решение:
   → «М250 отправлен обратно. М300 ожидаем к 14:00» 📌

5. Итого: проблема решена за 15 минут,
   переписка в треде — не засоряет основной чат
```

### Сценарий 2: Координация через видеозвонок

```
1. Руководитель проекта создаёт видеозвонок:
   → Тема: «Еженедельная планёрка — ЖК Южный парк»
   → Участники: прораб, снабженец, инженер ПТО

2. Все подключаются → обсуждают:
   → Прогресс за неделю
   → Прораб показывает экран с графиком
   → Решают: сдвинуть монтаж фасада на 3 дня

3. После звонка — длительность: 25 мин.
   → В истории звонков видно: кто участвовал, сколько длился

4. Руководитель пишет в канал:
   → «Итоги планёрки: фасад сдвигаем на 3 дня,
      монтаж окон начинаем по плану» 📌
```

### Сценарий 3: Избранное как личная записная книжка

```
1. В общем чате промелькнуло важное сообщение:
   → «Новые нормативы по огнезащите вступают в силу
     с 01.07.2026. Все проекты после этой даты —
     по новым требованиям.»

2. Инженер добавляет в избранное:
   → Заметка: «Проверить проект Южный парк —
     сдача в августе, нужно перейти на новые нормы»

3. Через месяц — открывает избранное:
   → Находит заметку → вовремя обновляет проект
```

## Частые вопросы

**Чем мессенджер отличается от почты?**
Мессенджер — для **быстрого внутреннего** общения (как чат). Почта — для **внешней официальной** переписки (с заказчиками, поставщиками, госорганами).

**Видят ли мои личные сообщения другие люди?**
Нет. Личные сообщения видят только два участника разговора. Администратор системы тоже не видит содержимое личных сообщений.

**Можно ли удалить отправленное сообщение?**
Да, но оно удалится у всех участников канала. Удаление необратимо.

**Как найти старое сообщение?**
Используйте поиск (🔍) в канале. Также важные сообщения можно заранее добавлять в избранное или закреплять.

**Можно ли пригласить гостя на звонок?**
Да. При создании звонка генерируется **ссылка-приглашение**. Гость подключается без аккаунта — просто вводит своё имя.

**Работают ли уведомления в статусе «Не беспокоить»?**
Нет. В этом статусе все уведомления от мессенджера отключены. Сообщения будут ждать вас, когда вы вернётесь.

**Хранятся ли файлы, отправленные в чат?**
Да, все файлы сохраняются. Вкладка «Файлы» в информации о канале показывает все прикреплённые файлы.
