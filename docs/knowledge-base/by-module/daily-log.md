# Ежедневный журнал (Daily Log)

---

## Что это и зачем

**Ежедневный журнал** — это записи прораба о том, что произошло на объекте за день: какие работы велись, погода, люди, техника, проблемы. Это основа для КС-6а и, в итоге, для КС-2.

Без журнала:
- Невозможно подтвердить объёмы выполненных работ
- При спорах с заказчиком нет доказательств
- Данные для КС-2 собираются «по памяти» — с ошибками

## Часть 1: Запись за день

| Раздел | Что записывается |
|--------|-----------------|
| **Дата** | Какой день |
| **Проект** | На каком объекте |
| **Погода** | Температура, осадки, ветер (влияет на возможность работ) |
| **Работы** | Перечень выполненных работ с объёмами |
| **Бригады** | Какие бригады работали, сколько человек |
| **Техника** | Какая техника была задействована |
| **Материалы** | Какие материалы израсходованы |
| **Проблемы** | Простои, задержки, инциденты |
| **Фото** | Фотофиксация хода работ |
| **Посетители** | Кто был на объекте (инспекторы, заказчик) |

### Статусы

| Статус | Описание |
|--------|----------|
| **Черновик** | Заполняется в течение дня |
| **Завершён** | Прораб подтвердил — запись закрыта |
| **Утверждён** | Руководитель проверил |

---

## Часть 2: Связь с КС-6а

Данные из ежедневного журнала аккумулируются в **журнал КС-6а** — учёт выполненных работ за период. Из КС-6а формируется **КС-2** (акт выполненных работ).

```
Ежедневный журнал (каждый день)
  → КС-6а (накопительный за период)
    → КС-2 (акт за месяц)
      → КС-3 (справка о стоимости)
        → Счёт → Оплата
```

---

## Типичный сценарий

```
1. Утро, 08:00. Прораб открывает журнал:
   → Дата: 15.03.2026
   → Проект: ЖК «Южный парк»
   → Погода: −3°C, ясно, ветер 5 м/с

2. В течение дня записывает:
   → Бетонирование перекрытия 3 эт. — 28 м³
   → Армирование колонн 4 эт. — 4 200 кг
   → Бригада Иванова: 8 чел., бригада Петрова: 6 чел.
   → Автобетононасос: 8 часов
   → Простой 2 часа: ждали доставку арматуры

3. Вечер: загружает 5 фотографий → «Завершено»

4. В конце месяца:
   → Все записи → КС-6а → КС-2 ✅
```

## Частые вопросы

**Обязательно ли вести журнал каждый день?**
Настоятельно рекомендуется. Без ежедневных записей невозможно корректно сформировать КС-2 и подтвердить объёмы перед заказчиком.

**Можно ли редактировать запись за прошлый день?**
Черновик — да. Завершённую запись — только с разрешения руководителя (все изменения логируются).
