# Первоначальная настройка системы

---

## Что это и зачем

**Первоначальная настройка** — пошаговое руководство для запуска системы ПРИВОД в вашей компании. От регистрации до первого рабочего проекта.

---

## Порядок настройки

### Шаг 1: Регистрация и вход

1. Перейдите на сайт системы
2. Нажмите «Регистрация»
3. Заполните: название компании, ИНН, ФИО руководителя, email, телефон
4. Подтвердите email (придёт письмо)
5. Войдите в систему

> Первый пользователь автоматически получает роль **Администратор**.

---

### Шаг 2: Настройки компании

**Путь:** Настройки → Компания

| Что заполнить | Зачем |
|---------------|-------|
| **Реквизиты** | ИНН, КПП, ОГРН, юр. адрес — для документов (КС-2, счетов) |
| **Банковские реквизиты** | Р/с, банк, БИК — для платёжных документов |
| **Логотип** | Будет на документах и в интерфейсе |
| **Рабочие часы** | График работы (для расчёта табеля) |
| **Часовой пояс** | Для корректного времени |

---

### Шаг 3: Создание пользователей

**Путь:** Настройки → Пользователи → Добавить

| Поле | Описание |
|------|----------|
| **ФИО** | Полное имя сотрудника |
| **Email** | Для входа в систему |
| **Роль** | Администратор / Руководитель / Инженер / Бухгалтер / Наблюдатель |
| **Отдел** | К какому подразделению относится |

Рекомендуемый порядок:
1. Сначала — руководители (директор, ГИП, главный инженер)
2. Затем — ключевые специалисты (сметчик, бухгалтер, снабженец)
3. Далее — остальные по мере необходимости

> **Совет:** Не создавайте всех сразу. Начните с 5-10 ключевых пользователей, освойтесь, потом подключайте остальных.

---

### Шаг 4: Справочники

**Путь:** Настройки → Справочники

Заполните базовые справочники:

| Справочник | Что внести | Примечание |
|-----------|-----------|------------|
| **Подразделения** | Отделы компании | ПТО, бухгалтерия, снабжение... |
| **Должности** | Типовые должности | Прораб, инженер ПТО, сметчик... |
| **Виды работ** | Типы строительных работ | СМР, электромонтаж, сантехника... |
| **Единицы измерения** | м², м³, шт., т, п.м. | Основные уже заполнены |
| **Статьи затрат** | Статьи бюджета | Материалы, работа, техника... |

> Можно загрузить из Excel — см. [Обмен данными](data-exchange.md).

---

### Шаг 5: Контрагенты

**Путь:** CRM → Контрагенты → Добавить

Внесите основных партнёров:

| Кого внести | Зачем |
|------------|-------|
| **Заказчики** | Для создания проектов и договоров |
| **Поставщики** | Для закупок материалов |
| **Субподрядчики** | Для конкурентных листов и договоров |
| **Банки** | Для платёжных реквизитов |

> Можно импортировать из Excel или 1С — см. [Обмен данными](data-exchange.md).

---

### Шаг 6: Первый проект

**Путь:** Проекты → Создать проект

| Что заполнить | Описание |
|---------------|----------|
| **Название** | Например: «ЖК Южный, корпус 3» |
| **Заказчик** | Из справочника контрагентов |
| **Адрес объекта** | Для привязки на карте |
| **Даты** | Плановое начало и окончание |
| **Бюджет** | Плановая стоимость |

При создании проекта автоматически создаётся:
- Финансовая модель (ФМ)
- Структура папок для документов
- Шаблон бюджета

---

### Шаг 7: Загрузка данных

Теперь можно загружать рабочие данные:

```
1. Загрузите смету (ЛСР из ГРАНД-Сметы) → попадёт в ФМ
2. Создайте спецификацию → позиции пойдут в закупки
3. Заведите договор с заказчиком
4. Назначьте команду проекта
5. Система готова к работе!
```

## Частые вопросы

**Сколько времени занимает настройка?**
Базовая настройка (шаги 1-5) — 1-2 часа. Загрузка данных по первому проекту — ещё 2-3 часа. Через день можно начинать полноценную работу.

**Можно ли перенести данные из другой системы?**
Да. Из Excel — через импорт (XLSX/CSV). Из 1С — через модуль обмена (XML). Из других систем — через CSV-экспорт.

**Нужен ли IT-специалист для настройки?**
Нет. Система облачная — ничего устанавливать не нужно. Настройка через интерфейс, без программирования. Но если у вас сложная структура (филиалы, нестандартные роли) — обратитесь в поддержку.

**Можно ли попробовать бесплатно?**
Да, есть бесплатный тариф — см. [Тарифы и подписка](subscription.md).
