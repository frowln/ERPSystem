# Материалы (Номенклатура)

---

## Что это и зачем

Материал — это любой строительный ресурс, который учитывается на складе: кирпич, цемент, арматура, краска, крепёж, инструмент и т.д. Каждый материал имеет наименование, код, единицу измерения и категорию.

Справочник материалов — это основа складского учёта. Пока материал не добавлен в справочник, его нельзя:
- Принять на склад
- Выдать на объект
- Включить в заявку на закупку
- Учесть в отчёте М-29
- Отсканировать штрих-кодом

**Проще говоря:** сначала создайте материал в справочнике — потом работайте с ним на складе.

---

## Где найти

**Путь:** Главное меню → **Склад** → **Материалы**

---

## Список материалов

**Что видно в таблице:**

| Колонка | Что показывает |
|---------|---------------|
| Код | Внутренний артикул (например, «МТ-001») |
| Наименование | Полное название (жирным) |
| Категория | Цветной бейдж: Бетон, Металл, Дерево и т.д. |
| Ед. изм. | Единица измерения (шт, м, кг, л) |
| Текущая цена | Последняя известная цена за единицу (₽) |

**Фильтры:**
- **Поиск** (строка ввода) — по коду и наименованию, фильтрует в реальном времени
- **Категория** (выпадающий список) — показать только одну категорию или все

**Действия:**
- Нажмите на строку → откроется карточка материала
- Кнопка **«+ Новый материал»** (сверху) → форма создания

---

## Категории материалов

В системе 10 категорий. Каждый материал обязательно относится к одной из них:

| Категория | Что входит | Цвет бейджа |
|-----------|-----------|-------------|
| **Бетон и растворы** | Бетон, раствор, пескобетон, грунтовка | Серый |
| **Металлоконструкции** | Арматура, балки, профиль, листовой металл | Синий |
| **Дерево и пиломатериалы** | Доска, брус, фанера, ОСП | Коричневый |
| **Изоляция** | Минвата, пенопласт, пеноплекс, мембраны | Оранжевый |
| **Трубы и фитинги** | ПВХ, металлические, ПНД; фитинги, задвижки | Фиолетовый |
| **Электрика** | Кабель, автоматы, розетки, светильники | Жёлтый |
| **Отделочные материалы** | Краска, штукатурка, плитка, ламинат | Розовый |
| **Крепёж** | Болты, гайки, саморезы, анкера, дюбели | Красный |
| **Инструменты** | Ручной и электроинструмент (как расходники) | Зелёный |
| **Прочее** | Всё, что не вошло в другие категории | Нейтральный |

## Единицы измерения

| Ед. изм. | Полное название | Для чего |
|----------|----------------|---------|
| **шт** | Штуки | Кирпичи, болты, светильники |
| **м** | Метры | Арматура, труба, кабель |
| **м²** | Квадратные метры | Фанера, плитка, утеплитель |
| **м³** | Кубические метры | Бетон, щебень, песок |
| **кг** | Килограммы | Цемент, краска, клей |
| **т** | Тонны | Арматура оптом, бетон, щебень |
| **л** | Литры | Краска, грунтовка, масло |
| **упак** | Упаковки | Саморезы в коробке, плитка в пачке |
| **комплект** | Комплекты | Электрощит в сборе, набор инструментов |

## Как создать материал

1. Перейдите в **Склад → Материалы**
2. Нажмите **«+ Новый материал»**

3. Заполните форму:

### Основные данные

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Код** | Да | Внутренний артикул, до 100 символов. Должен быть уникальным в организации. Пример: «МТ-001» |
| **Наименование** | Да | Полное название, до 500 символов. Пример: «Арматура А500С d12 мм» |
| **Категория** | Да | Выберите из 10 категорий |
| **Ед. изм.** | Да | Выберите единицу измерения |

### Складское хранение

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Мин. остаток** | Нет | Ниже какого количества — сигнал «Заказать». По умолчанию 0 |
| **Место хранения** | Нет | Текстовое описание, где обычно лежит. Пример: «Склад А, стеллаж 3, полка 2» |

4. Нажмите **«Создать материал»** — появится уведомление «Материал создан», вы вернётесь в список.

> **Важно:** Код должен быть уникальным. Если код уже занят — система покажет ошибку.

## Карточка материала (детальная страница)

При нажатии на строку в списке открывается полная карточка.

**Заголовок:**
- Название материала (крупно)
- Код / Категория (бейдж)
- Если остаток ниже минимума → красный бейдж **«Требуется заказ»**

**Показатели:**

| Показатель | Что значит |
|-----------|-----------:|
| Общий остаток | Сколько всего на всех складах (число + ед. изм.) |
| Мин. остаток | Порог, ниже которого нужно заказывать |
| Макс. остаток | Верхняя граница (если задана) |
| Средняя цена | Последняя цена за единицу |

**Остатки по складам (таблица):**

| Склад | Всего | Зарезервировано | Доступно |
|-------|------:|----------------:|---------:|
| Центральный склад | 100 шт | 20 шт | 80 шт |
| Площадка «Дом на Ленина» | 150 шт | 0 шт | 150 шт |

> **Зарезервировано** — материал уже выделен для проекта, но ещё не выдан. Он числится на складе, но недоступен для новых заказов.

**Последние движения:**
Список из 10 последних операций: тип (приход/расход/перемещение), откуда → куда, количество, дата.

**Кнопки:**
- **«Редактировать»** — изменить данные материала
- **«Удалить»** — мягкое удаление (материал скрывается, но остаётся в истории)
- **«Создать заявку на поставку»** — если остаток низкий, быстро перейти в закупки

## Штрих-код

У каждого материала может быть штрих-код (или QR-код) — необязательное поле.

**Как работает:**
1. Штрих-код присваивается материалу (вводится вручную или из наклейки)
2. На складе кладовщик сканирует штрих-код мобильным устройством
3. Система мгновенно находит материал по коду
4. Автоматически заполняет формы прихода, расхода или инвентаризации

> Штрих-код должен быть уникальным в пределах организации. Подробнее о сканере — в статье [Складской учёт](warehouse.md#штрих-код-сканер).

## Минимальный остаток и оповещения

Если задать **мин. остаток** (например, 50 м для арматуры), система будет следить:

```
Текущий остаток: 45 м
Минимальный остаток: 50 м
→ Остаток НИЖЕ минимума!
→ На карточке: красный бейдж «Требуется заказ»
→ На странице оповещений: предупреждение
→ Кнопка: «Создать заявку на поставку»
```

Когда поступит новая партия (например, +100 м), остаток станет 145 м — оповещение снимется автоматически.

Подробнее о лимитах и оповещениях — в статье [Складской учёт](warehouse.md#лимиты-запасов-и-оповещения).

---

## Мягкое удаление

Материалы **никогда** не удаляются из базы данных полностью. При нажатии «Удалить»:
- Материал помечается как удалённый
- Он исчезает из списка и из выпадающих списков
- Но все исторические данные сохраняются: движения, проводки, отчёты

Это нужно для аудита и проверок: если арматура числилась на складе год назад — данные о ней можно найти.

## Частые вопросы

**Как найти материал по штрих-коду?**
Перейдите в **Склад → Сканер штрих-кодов** и отсканируйте код. Система мгновенно покажет карточку материала с остатками. Подробнее — [Штрих-код сканер](warehouse.md#штрих-код-сканер).

**Можно ли добавить свою категорию?**
Нет. Категории фиксированы (10 штук). Если материал не подходит ни к одной — выберите «Прочее».

**Что значит «Зарезервировано»?**
Это материал, уже выделенный для конкретного проекта, но ещё не выданный физически. Он на складе, но недоступен для других проектов. Доступный остаток = Всего − Зарезервировано.

**Как посмотреть, где лежит материал?**
В карточке материала — таблица «Остатки по складам». Также можно использовать [Адресное хранение](warehouse.md#адресное-хранение) — там видно стеллаж, полку и ячейку.

**Можно ли восстановить удалённый материал?**
Удаление — мягкое (материал скрывается, но не удаляется из базы). Для восстановления обратитесь к администратору.

**Как обновляется цена?**
Цена обновляется автоматически при каждом приходе: если поступила партия арматуры по 55 ₽/кг, цена в карточке станет 55 ₽/кг. Это «последняя цена поступления», а не средняя.

**Можно ли загрузить материалы из Excel?**
Массовая загрузка через модуль «Импорт данных» (**Обмен данными → Импорт**). Формат файла: код, наименование, категория, единица, цена.

**Чем отличается «Материал» от «Позиция спецификации»?**
Материал — это запись в справочнике склада (общая для всей организации). Позиция спецификации — это строка в техдокументации конкретного проекта, которая ссылается на материал + указывает нужное количество.
