# Автопарк

---

## Что это и зачем

**Автопарк** — это модуль учёта строительной техники, транспорта и оборудования. Здесь вы ведёте всё: от самосвалов и кранов до служебных автомобилей.

Без модуля автопарка:
- Никто не знает, где сейчас кран — на объекте или в ремонте
- Путевые листы заполняют на бумаге, потом теряют
- Расход топлива никто не контролирует — списывают «на глаз»
- Техобслуживание пропускают — техника ломается в самый неподходящий момент
- Невозможно посчитать стоимость эксплуатации единицы техники

Модуль «Автопарк» в системе ПРИВОД:
- Реестр всей техники с характеристиками и статусами
- Путевые листы (форма ЭСМ-2) с предрейсовым контролем
- Учёт ГСМ: заправки, расход, нормы
- Плановое ТО по пробегу, моточасам или дате
- GPS-отслеживание на карте
- Рейтинг водителей

## Часть 2: Создание карточки техники

**Путь:** Техника → **«+ Новая единица»**

### Основные поля

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Название** | Да | Например: «КамАЗ-6520 самосвал» |
| **Тип** | Да | Из списка выше (грузовик, экскаватор...) |
| **Гос. номер** | Нет | Регистрационный знак (А123БВ77) |
| **VIN** | Нет | Идентификационный номер (17 символов) |
| **Год выпуска** | Нет | Год производства |
| **Модель** | Нет | Марка и модель |
| **Тип топлива** | Да | Дизель / Бензин / Газ / Электро / Гибрид |
| **Вместимость бака** | Нет | В литрах |
| **Норма расхода** | Нет | Литров на 100 км (или на моточас) |
| **Текущий пробег** | Нет | В километрах |
| **Моточасы** | Нет | Для техники без одометра |
| **Привязка к проекту** | Нет | На каком объекте работает |

### Документы техники

К карточке можно прикрепить:
- ПТС (паспорт транспортного средства)
- Свидетельство о регистрации
- Страховой полис (ОСАГО/КАСКО)
- Диагностическая карта
- Разрешение на работу крана и т.д.

## Часть 3: Статусы техники

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Активна** (ACTIVE) | Зелёный | Техника работает, в эксплуатации |
| **На обслуживании** (MAINTENANCE) | Жёлтый | Плановое ТО или текущий ремонт |
| **В ремонте** (REPAIR) | Оранжевый | Серьёзная поломка, ремонт |
| **На консервации** (STORAGE) | Серый | Не используется (сезонная, резервная) |
| **Списана** (DECOMMISSIONED) | Красный | Выведена из эксплуатации |

## Часть 4: Путевые листы (ЭСМ-2)

**Путь:** Техника → **Путевые листы**

Путевой лист — обязательный документ для каждого выезда техники. Без него выезд незаконен.

### Поля путевого листа

| Поле | Описание |
|------|----------|
| **Номер** | Автоматический (WB-001, WB-002...) |
| **Единица техники** | Какая машина выезжает |
| **Водитель** | Кто управляет |
| **Дата выдачи** | Когда выписан |
| **Дата возврата** | Когда техника вернулась |
| **Маршрут** | Откуда — куда |
| **Пробег на выезде** | Показания одометра при выезде |
| **Пробег на возврате** | Показания одометра при возврате |
| **Топливо выдано** | Сколько литров залили |
| **Топливо остаток** | Сколько осталось при возврате |

### Статусы путевого листа

| Статус | Описание |
|--------|----------|
| **Черновик** (DRAFT) | Создан, но ещё не выписан |
| **Выдан** (ISSUED) | Путевой лист выписан, техника на маршруте |
| **Завершён** (COMPLETED) | Рейс окончен, данные внесены |
| **Проверен** (VERIFIED) | Механик/диспетчер проверил |
| **Отменён** (CANCELLED) | Рейс не состоялся |

### Предрейсовый контроль

Перед выездом водитель (или механик) проходит **чек-лист**:
- Тормозная система — исправна ✅
- Рулевое управление — исправно ✅
- Освещение — работает ✅
- Шины — в норме ✅
- Уровень масла — в норме ✅
- Медосмотр водителя — пройден ✅

Если хотя бы один пункт не пройден — выезд запрещён.

## Часть 5: Учёт ГСМ

**Путь:** Техника → **Учёт ГСМ**

ГСМ — горюче-смазочные материалы (топливо, масло и т.д.).

### Что фиксируется

| Поле | Описание |
|------|----------|
| **Единица техники** | Какую машину заправили |
| **Дата** | Когда заправка |
| **Тип топлива** | ДТ / АИ-92 / АИ-95 / Газ |
| **Количество (л)** | Сколько залили |
| **Стоимость** | Сколько заплатили |
| **Цена за литр** | Автоматически (стоимость ÷ кол-во) |
| **Пробег** | Показания одометра на момент заправки |
| **Заправка** | АЗС или склад ГСМ |

### Контроль расхода

Система автоматически считает:
- **Фактический расход** — литров на 100 км (или на моточас)
- **Нормативный расход** — сколько должно было уйти по нормам
- **Отклонение** — перерасход или экономия

Если отклонение больше 15% — система выделяет строку красным. Это может означать:
- Водитель сливает топливо
- Техника неисправна (повышенный расход)
- Норма устарела (нужно пересмотреть)

## Часть 6: Техобслуживание и ремонт

**Путь:** Техника → **Обслуживание**

### Типы записей

| Тип | Описание | Пример |
|-----|----------|--------|
| **Плановое ТО** (SCHEDULED) | По графику | Замена масла каждые 10 000 км |
| **Текущий ремонт** (REPAIR) | Мелкая поломка | Замена тормозных колодок |
| **Капитальный ремонт** (OVERHAUL) | Крупный ремонт | Ремонт двигателя |
| **Диагностика** (INSPECTION) | Проверка состояния | Предсезонный осмотр |

### Статусы

| Статус | Описание |
|--------|----------|
| **Запланировано** (PLANNED) | В плане, ещё не начато |
| **В работе** (IN_PROGRESS) | Техника в сервисе |
| **Завершено** (COMPLETED) | Ремонт/ТО окончено |
| **Отменено** (CANCELLED) | Не потребовалось |

### График ТО

**Путь:** Техника → **График ТО**

Можно задать **правила планового ТО** с тремя типами триггеров:

| Триггер | Описание | Пример |
|---------|----------|--------|
| **По пробегу** | Каждые N километров | ТО каждые 15 000 км |
| **По моточасам** | Каждые N моточасов | Замена фильтров каждые 500 мч |
| **По дате** | Каждые N дней/месяцев | Проверка каждые 6 месяцев |

Типы работ в графике:
- Замена масла и фильтров
- Проверка тормозной системы
- Замена ремней
- Шиномонтаж (сезонный)
- Диагностика электрики
- Смазка узлов
- Проверка гидравлики
- Замена жидкостей (антифриз, тормозная)
- Общий технический осмотр

Когда подходит срок — система заранее уведомляет ответственного.

## Часть 7: GPS-трекинг

**Путь:** Техника → **GPS-трекинг**

На карте видно **все единицы техники** в реальном времени:

| Статус на карте | Значок | Описание |
|----------------|--------|----------|
| **В движении** | 🟢 | Техника едет |
| **Стоит** | 🟡 | Техника на месте, двигатель работает |
| **Припаркована** | ⚪ | Двигатель заглушен |
| **Нет сигнала** | 🔴 | GPS-трекер не передаёт данные |

### Что видно на карте
- Местоположение каждой единицы
- Скорость движения
- Направление
- Маршрут за день (трек)
- Геозоны (объекты, базы, склады)

### Рейтинг водителей

Система автоматически оценивает водителей по 4 параметрам:

| Параметр | Что оценивает |
|----------|--------------|
| **Расход топлива** | Экономичность вождения |
| **Стиль вождения** | Резкие ускорения, торможения |
| **Соблюдение маршрута** | Отклонения от заданного маршрута |
| **Техническое состояние** | Своевременные осмотры, жалобы на технику |

Рейтинг помогает определить лучших водителей и тех, кому нужно дополнительное обучение.

## Типичные сценарии

### Сценарий 1: Утренний выезд техники на объект

```
1. Диспетчер утром создаёт путевые листы:
   → КамАЗ-6520 (водитель Петров) — на объект «ЖК Южный парк»
   → Экскаватор JCB (машинист Сидоров) — на объект «ЖК Южный парк»

2. Механик проводит предрейсовый контроль:
   → КамАЗ: все пункты ✅ → допущен к выезду
   → Экскаватор: течь гидравлики ❌ → выезд запрещён!

3. КамАЗ уезжает → статус путевого листа: Выдан
   Экскаватор → запись на ремонт → статус техники: В ремонте

4. Вечером КамАЗ возвращается:
   → Пробег на выезде: 45 230 км
   → Пробег на возврате: 45 312 км
   → Проехал: 82 км
   → Расход: 28 л (норма 32 л на 100 км = 26,2 л) — перерасход 7%
   → Статус путевого листа: Завершён ✅
```

### Сценарий 2: Контроль расхода ГСМ

```
1. Главный механик смотрит отчёт по ГСМ за месяц:
   → КамАЗ-001: расход 34 л/100км (норма 32) — +6% ✅
   → КамАЗ-002: расход 41 л/100км (норма 32) — +28% ❌❗

2. КамАЗ-002 — красная строка в отчёте
   → Механик проверяет технику → негерметичность топливной системы
   → Отправляет на ремонт → создаёт запись «Текущий ремонт»

3. После ремонта:
   → Расход вернулся к норме: 33 л/100 км
   → Экономия: ~200 литров в месяц
```

### Сценарий 3: Плановое ТО по пробегу

```
1. Система уведомляет:
   «КамАЗ-6520 — до планового ТО осталось 500 км»

2. Механик планирует ТО на выходные:
   → Создаёт запись: Плановое ТО
   → Работы: замена масла, фильтров, проверка тормозов
   → Статус техники: На обслуживании

3. В субботу:
   → ТО выполнено
   → Запись: Завершено
   → Стоимость: 15 000 руб.
   → Следующее ТО: через 15 000 км

4. В понедельник:
   → Статус техники: Активна
   → КамАЗ снова на линии
```

## Частые вопросы

**Обязательно ли вести путевые листы?**
Да. По закону каждый выезд коммерческого транспорта должен сопровождаться путевым листом. Штраф за отсутствие — до 30 000 руб.

**Как считается расход топлива?**
Автоматически: (залито – остаток) ÷ пробег × 100. Или по данным GPS-трекера и бортового компьютера.

**Можно ли вести технику без GPS?**
Да. GPS-трекинг — дополнительная функция. Базовый учёт (путевые, ГСМ, ТО) работает без GPS.

**Что делать, если техника арендованная?**
Ведите её так же, как свою. Можно отметить в примечании «Аренда у ООО ...». Учёт ГСМ и путевые листы нужны в любом случае.

**Как настроить нормы расхода?**
В карточке техники укажите «Норма расхода» в литрах на 100 км. Нормы берутся из приказа Минтранса (распоряжение АМ-23-р) или из технического паспорта.
