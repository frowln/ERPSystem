# Подписки и оплата

---

## Что это и зачем

**Подписка** определяет, какие функции системы ПРИВОД вам доступны. Есть три тарифных плана: бесплатный, профессиональный и корпоративный.

---

## Где найти

**Путь:** Главное меню → **Настройки** → **Подписка**

| Страница | URL | Что делает |
|----------|-----|-----------|
| **Тарифы** | /settings/subscription | Текущий план и доступные тарифы |
| **Результат оплаты** | /settings/subscription/result | Статус платежа |

---

## Часть 1: Тарифные планы

| | **Бесплатный** (FREE) | **Профессиональный** (PRO) | **Корпоративный** (ENTERPRISE) |
|--|----------------------|---------------------------|-------------------------------|
| **Цена** | 0 ₽/мес. | от 5 000 ₽/мес. | Индивидуально |
| **Пользователи** | до 3 | до 50 | Без ограничений |
| **Проекты** | до 2 | до 100 | Без ограничений |
| **Хранилище** | 1 ГБ | 100 ГБ | Без ограничений |
| **Модули** | Базовые | Все | Все + персонализация |
| **Поддержка** | Тикеты | Приоритетная | Выделенный менеджер |
| **Интеграции** | ❌ | ✅ 1С, ИСУП, Telegram | ✅ + кастомные |
| **BIM** | ❌ | ✅ | ✅ |
| **Аналитика** | Базовая | Полная | Полная + BI |
| **API** | ❌ | ✅ | ✅ + webhook |
| **Резервное копирование** | Ежедневно | Ежедневно | По расписанию |
| **2FA** | ❌ | ✅ | ✅ |

### Бесплатный план (FREE)

Подходит для:
- Знакомства с системой
- Маленьких компаний (1–3 человека)
- Одного-двух проектов

Ограничения:
- Максимум 3 пользователя
- Максимум 2 активных проекта
- 1 ГБ хранилища для файлов
- Нет интеграций (1С, ИСУП, Telegram)
- Нет BIM и расширенной аналитики

### Профессиональный план (PRO)

Подходит для:
- Строительных компаний среднего размера
- Нескольких проектов одновременно
- Нужна интеграция с 1С

Включает всё из бесплатного, плюс:
- До 50 пользователей
- До 100 проектов
- 100 ГБ хранилища
- Все модули без ограничений
- Интеграции: 1С, ИСУП, Telegram, ЭДО
- BIM, полная аналитика, конструктор отчётов
- Двухфакторная аутентификация
- Приоритетная поддержка

### Корпоративный план (ENTERPRISE)

Подходит для:
- Крупных строительных холдингов
- Десятков проектов и сотен сотрудников
- Нужна персонализация и выделенная поддержка

Включает всё из PRO, плюс:
- Без ограничений по пользователям и проектам
- Без ограничений по хранилищу
- Кастомные интеграции
- Выделенный менеджер поддержки
- Персонализация интерфейса
- Установка на собственный сервер (on-premise)
- SLA с гарантированным временем реакции

## Часть 2: Оплата

### Способы оплаты

Оплата принимается через **ЮKassa** (бывшие Яндекс.Деньги):
- Банковская карта (Visa, MasterCard, МИР)
- Электронный кошелёк (ЮMoney)
- Банковский перевод (для юридических лиц)

### Процесс оплаты

1. Выберите тарифный план
2. Нажмите **«Оплатить»**
3. Система перенаправит на страницу ЮKassa
4. Введите данные карты и подтвердите
5. После оплаты — автоматическое переключение тарифа

### Периоды оплаты

| Период | Скидка |
|--------|--------|
| **Ежемесячно** | Базовая цена |
| **Ежегодно** | Скидка 20% |

## Часть 3: Квоты и лимиты

Каждый тариф имеет **квоты** — лимиты использования:

| Квота | FREE | PRO | ENTERPRISE |
|-------|------|-----|------------|
| Пользователи | 3 | 50 | ∞ |
| Активные проекты | 2 | 100 | ∞ |
| Хранилище (ГБ) | 1 | 100 | ∞ |

### Что происходит при превышении квоты

- **Пользователи**: нельзя создать нового пользователя — система предложит повысить тариф
- **Проекты**: нельзя создать новый проект — нужно завершить существующий или повысить тариф
- **Хранилище**: нельзя загрузить файл — нужно удалить старые файлы или повысить тариф

> Существующие данные **не удаляются** при превышении квоты. Вы просто не сможете добавить новые.

## Часть 4: История платежей

На странице подписки видна **история платежей**:

| Поле | Описание |
|------|----------|
| **Дата** | Когда был платёж |
| **Сумма** | Сколько оплачено |
| **Тариф** | За какой план |
| **Период** | За какой период (месяц/год) |
| **Статус** | Оплачен / Ожидает / Отменён |
| **Чек** | Ссылка на электронный чек |

## Типичные сценарии

### Сценарий 1: Переход с бесплатного на PRO

```
1. Компания начала с бесплатного плана:
   → 2 пользователя, 1 проект — хватает

2. Получили новый контракт — нужен 3-й проект:
   → Система: «Лимит проектов исчерпан.
     Перейдите на план PRO.»

3. Директор переходит:
   → Настройки → Подписка → PRO
   → Ежемесячная оплата: 5 000 руб.
   → Оплата картой через ЮKassa ✅

4. Мгновенно:
   → Лимит проектов: 100
   → Лимит пользователей: 50
   → Хранилище: 100 ГБ
   → Доступны интеграции, BIM, аналитика
```

### Сценарий 2: Ежегодное продление

```
1. Бухгалтер получает email за 30 дней до окончания:
   «Ваша подписка PRO истекает 01.05.2026.
    Продлите, чтобы не потерять доступ.»

2. Бухгалтер:
   → Настройки → Подписка → «Продлить»
   → Выбирает: годовая оплата (скидка 20%)
   → 5 000 × 12 × 0.8 = 48 000 руб./год (вместо 60 000)
   → Оплата по счёту (банковский перевод)

3. После оплаты:
   → Подписка продлена до 01.05.2027 ✅
   → Чек доступен в истории платежей
```

## Частые вопросы

**Что будет, если не продлить подписку?**
Система перейдёт на бесплатный план. Данные **не удалятся**, но функции будут ограничены. Если у вас больше 3 пользователей или 2 проектов — новые создать не получится, но существующие останутся.

**Можно ли вернуть деньги?**
Возврат возможен в течение 14 дней с момента оплаты, если вы не использовали платные функции.

**Есть ли пробный период?**
Да. При регистрации автоматически включается **14-дневный пробный период** плана PRO. После окончания — переход на бесплатный или оплата.

**Можно ли платить по счёту (безналичный расчёт)?**
Да, для юридических лиц. Выберите «Банковский перевод» при оплате. Счёт формируется автоматически.

**Как узнать, сколько осталось квоты?**
На странице подписки отображаются текущие значения: сколько пользователей создано из лимита, сколько проектов, сколько занято хранилища.

**Можно ли понизить тариф?**
Да. Но если текущее использование превышает лимиты нового плана — нужно сначала привести в соответствие (уменьшить количество пользователей, проектов или файлов).
