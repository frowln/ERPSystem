# Кадровые формы РФ

---

## Что это и зачем

Модуль «Кадры РФ» — это набор форм и документов, обязательных по Трудовому кодексу Российской Федерации. Каждое кадровое действие (приём, увольнение, перевод, отпуск, командировка) должно быть оформлено соответствующим документом. Без этих документов — штрафы от трудовой инспекции.

В системе 7 разделов:

| Раздел | Что оформляет |
|--------|-------------|
| **Трудовые договоры** | Соглашение между работодателем и сотрудником |
| **Кадровые приказы** | Официальные распоряжения (приём, увольнение, перевод) |
| **Штатное расписание** | Структура должностей и окладов |
| **Отпуска** | Все виды отпусков сотрудников |
| **Больничные листы** | Периоды нетрудоспособности |
| **Командировки** | Служебные поездки |
| **Табель Т-13** | Официальная форма учёта рабочего времени |

## 1. Трудовые договоры

Трудовой договор — главный документ между работодателем и сотрудником. Без него — нарушение ТК РФ (штраф до 100 000 руб.).

### Виды договоров

| Тип | Описание | Когда используется |
|-----|---------|-------------------|
| **Бессрочный** | Без даты окончания | Основная форма трудоустройства |
| **Срочный** | С конкретной датой окончания | Временные работы, замещение, проект |
| **ГПХ** | Гражданско-правовой характер | Подряд, разовые услуги |

### Поля договора

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Номер** | Да | Уникальный номер (например, «ТД-2026-001») |
| **Сотрудник** | Да | Кого принимаем |
| **Тип** | Да | Бессрочный / Срочный / ГПХ |
| **Дата начала** | Да | С какого числа |
| **Дата окончания** | Для срочного | До какого числа |
| **Должность** | Да | На какую позицию |
| **Отдел** | Нет | В каком подразделении |
| **Оклад** | Да | Размер зарплаты |
| **Тип оплаты** | Да | Оклад / Почасовая / Сдельная |
| **График работы** | Нет | Например: «5/2, 40 часов/нед.» |
| **Испытательный срок** | Нет | Дата окончания испытания (обычно 3 месяца) |

### Статусы договора

| Статус | Цвет | Описание |
|--------|------|---------|
| **Действующий** | Зелёный | Договор в силе |
| **Расторгнут** | Красный | Трудовые отношения прекращены |
| **Приостановлен** | Жёлтый | Временно приостановлен |
| **Истёк** | Серый | Срок срочного договора закончился |

### Автоматические проверки

- **Испытание:** Если `испытательный срок < сегодня` → автоматически снимается флаг «на испытании»
- **Срок:** Если `дата окончания < сегодня` → статус «Истёк»
- Система оповещает о договорах, срок которых скоро истекает

## 2. Кадровые приказы

Кадровый приказ — это внутренний распорядительный документ на каждое действие с сотрудником.

### Типы приказов

| Тип | Описание | Когда оформляется |
|-----|---------|------------------|
| **Приём** | Приказ о приёме на работу | При заключении трудового договора |
| **Увольнение** | Приказ об увольнении | При расторжении договора |
| **Перевод** | Приказ о переводе | Смена должности, отдела или условий |
| **Отпуск** | Приказ об отпуске | При утверждении отпуска |
| **Командировка** | Приказ о командировке | При направлении в поездку |

### Поля приказа

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Номер** | Да | Номер приказа (например, «ПР-2026-045») |
| **Сотрудник** | Да | Кого касается |
| **Тип** | Да | Приём / Увольнение / Перевод / Отпуск / Командировка |
| **Дата приказа** | Да | Когда подписан |
| **Дата вступления** | Да | С какого числа действует |
| **Основание** | Нет | На основании чего (статья ТК, заявление) |
| **Содержание** | Нет | Текст приказа |
| **Подписал** | Нет | Кто подписал (руководитель) |

### Статусы приказа

| Статус | Описание |
|--------|---------|
| **Черновик** | Подготовка |
| **На согласовании** | У руководителя |
| **Согласован** | Утверждён |
| **Исполнен** | Приказ выполнен |
| **Отменён** | Аннулирован |

```
Черновик → На согласовании → Согласован → Исполнен
                                          ↓
                                      Отменён
```

### Показатели на странице

- Всего приказов
- Приёмы за период
- Увольнения за период
- Ожидают согласования

## 3. Штатное расписание

Штатное расписание — документ, определяющий структуру организации: какие должности есть, сколько единиц, какой оклад.

### Поля

| Поле | Описание |
|------|---------|
| **Должность** | Название позиции |
| **Отдел** | Подразделение |
| **Грейд** | Уровень (Junior / Middle / Senior) |
| **Оклад (мин.)** | Нижняя граница зарплаты |
| **Оклад (макс.)** | Верхняя граница |
| **Штатных единиц** | Сколько человек на эту должность |
| **Занято** | Сколько фактически работает |
| **Вакансий** | Штатных − Занято |

### Показатели

| Показатель | Что значит |
|-----------|-----------:|
| Всего единиц | Общий штат по расписанию |
| Занято | Сколько человек работает |
| Вакансий | Сколько позиций открыто |
| Укомплектованность | Занято ÷ Всего × 100% |

### Статусы позиций

| Статус | Цвет | Описание |
|--------|------|---------|
| **Укомплектована** | Зелёный | Все единицы заняты |
| **Вакантна** | Синий | Есть свободные места |
| **Зарезервирована** | Жёлтый | Место под внутренний перевод |
| **Ликвидирована** | Серый | Должность убрана из штатного |

> **Важно:** Нельзя принять человека на должность, которой нет в штатном расписании.

## 4. Отпуска (кадровый учёт)

В модуле «Кадры РФ» отпуска ведутся параллельно с [основным модулем отпусков](leave.md), но с акцентом на кадровое оформление.

### Виды отпусков (РФ)

| Тип | Описание | Дни |
|-----|---------|-----|
| **Ежегодный** | Основной оплачиваемый | 28 (минимум по ТК) |
| **Дополнительный** | За вредность, ненормированный день | Зависит от условий |
| **Без содержания** | Без сохранения зарплаты | По заявлению |
| **Учебный** | Для сдачи экзаменов, защиты диплома | До 50 дней |
| **Декретный** | По беременности и родам | 140–160 дней |

### Статусы

```
Запланирован → Утверждён → Действует → Завершён
                                ↓
                            Отменён
```

### Поля

| Поле | Описание |
|------|---------|
| Сотрудник | Кто уходит |
| Тип | Вид отпуска |
| Дата начала — окончания | Период отпуска |
| Кол-во дней | Автоматический расчёт |
| Замещающий | Кто выполняет обязанности |
| Приказ | Ссылка на кадровый приказ об отпуске |

## 5. Больничные листы

Больничный лист — период нетрудоспособности, подтверждённый медицинской справкой.

### Поля

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Сотрудник** | Да | Кто болеет |
| **Дата начала** | Да | Первый день больничного |
| **Дата окончания** | Да (при закрытии) | Последний день |
| **Кол-во дней** | Авто | Рассчитывается |
| **Номер листа** | Нет | Номер больничного листа (электронный) |
| **Диагноз** | Нет | Если известен |
| **Продление** | Нет | Является ли продлением предыдущего |
| **Сумма выплаты** | Нет | Размер компенсации |

### Статусы

| Статус | Описание |
|--------|---------|
| **Открыт** | Сотрудник на больничном |
| **Закрыт** | Больничный завершён, сотрудник вышел на работу |
| **Аннулирован** | Отменён (ошибочно оформлен) |

### Оплата больничного (по ТК РФ)

```
Первые 3 дня — оплачивает работодатель
С 4-го дня — оплачивает ФСС (Фонд социального страхования)
Размер оплаты зависит от стажа:
  До 5 лет стажа — 60% от среднего заработка
  5–8 лет — 80%
  Более 8 лет — 100%
```

### Полезные фильтры

- Все открытые больничные → показывает, кто сейчас болеет
- По сотруднику → история болезней
- По периоду → статистика за месяц/квартал

## 6. Командировки

Командировка — служебная поездка сотрудника по распоряжению работодателя.

### Поля

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Сотрудник** | Да | Кто едет |
| **Пункт назначения** | Да | Куда (город, адрес) |
| **Цель** | Да | Зачем (встреча, монтаж, обучение) |
| **Дата начала** | Да | Дата выезда |
| **Дата окончания** | Да | Дата возвращения |
| **Кол-во дней** | Авто | Рассчитывается |
| **Суточные** | Нет | Размер суточных (₽/день) |
| **Общий бюджет** | Нет | Плановая стоимость поездки |
| **Приказ** | Нет | Ссылка на кадровый приказ |
| **Дата отчёта** | Нет | Когда сдал авансовый отчёт |

### Статусы

| Статус | Описание |
|--------|---------|
| **Запланирована** | Командировка оформлена |
| **Утверждена** | Руководитель одобрил |
| **Действует** | Сотрудник в поездке |
| **Завершена** | Вернулся, отчёт сдан |
| **Отменена** | Поездка не состоялась |

```
Запланирована → Утверждена → Действует → Завершена
                                 ↓
                             Отменена
```

### Суточные (по ТК РФ)

- По территории РФ: обычно 700 руб./день (не облагается НДФЛ)
- Заграничные: обычно 2 500 руб./день (не облагается НДФЛ)
- Точные суммы определяются приказом организации

## 7. Табель Т-13

Подробное описание — в статье [Табели](timesheets.md#часть-2-табель-т-13-официальная-форма).

Здесь — краткая сводка:

- Т-13 — официальная форма Госкомстата
- Сетка: строки = сотрудники, столбцы = дни месяца
- В каждой ячейке — код явки (Я, Б, О, К, НН, В, Н) и часы
- Итоги: дней, часов, ночных часов по каждому сотруднику
- Статусы: Черновик → На утверждении → Утверждён

## Личная карточка Т-2

Личная карточка — это стандартная форма, содержащая все данные о сотруднике. В системе хранится в JSON-формате.

**Содержит:**
- ФИО, дата рождения, место рождения
- Паспортные данные
- Адрес регистрации и фактический
- Контакты
- Образование
- Семейное положение
- Воинский учёт (для военнообязанных)

**Где найти:** В карточке [сотрудника](employees.md) — все данные из Т-2 распределены по вкладкам.

## Трудовая книжка

С 2020 года в России действуют электронные трудовые книжки.

**Поля в системе:**
- Серия и номер книжки
- Формат: бумажная или электронная
- Записи: история трудоустройства (дата, должность, отдел, основание)
- Дата последней записи

## Типичные кадровые процессы

### Приём на работу

```
1. Создать запись в штатном расписании (если должности нет)
2. Создать карточку сотрудника (ФИО, паспорт, ИНН, СНИЛС)
3. Создать трудовой договор (тип, оклад, дата начала, испытание)
4. Создать кадровый приказ (тип: «Приём»)
5. Назначить на проект / в бригаду
```

### Увольнение

```
1. Создать кадровый приказ (тип: «Увольнение», основание)
2. Расторгнуть трудовой договор (статус → «Расторгнут»)
3. Изменить статус сотрудника → «Уволен»
4. Снять с проектов и бригад
5. Рассчитать и выплатить компенсацию
```

### Перевод

```
1. Создать кадровый приказ (тип: «Перевод»)
2. Обновить должность/отдел в карточке сотрудника
3. При необходимости — дополнительное соглашение к договору
```

### Оформление отпуска

```
1. Сотрудник подаёт заявление на отпуск
2. Руководитель утверждает
3. HR создаёт кадровый приказ (тип: «Отпуск»)
4. В табеле Т-13 — код «О» за дни отпуска
```

## Частые вопросы

**Обязательно ли оформлять приказ на каждое действие?**
По ТК РФ — да. Приём, увольнение, перевод, отпуск, командировка — всё должно быть оформлено приказом. Без приказа действие юридически не оформлено.

**Чем отличается бессрочный договор от срочного?**
Бессрочный — без даты окончания, основная форма. Срочный — на определённый срок (например, на время проекта или замещения). По закону, срочный можно заключать только в определённых случаях.

**Что такое испытательный срок?**
Период, в течение которого работодатель проверяет квалификацию нового сотрудника. Обычно 3 месяца (для руководителей — до 6 месяцев). В период испытания можно уволить с упрощённой процедурой.

**Как оформить больничный?**
Создайте запись в разделе «Больничные листы» с датами и номером листа. Закройте, когда сотрудник вернулся. Оплата рассчитывается автоматически на основе стажа.

**Где взять номер больничного листа?**
Электронный больничный лист (ЭЛН) формируется врачом в системе ФСС. Номер сообщается сотруднику и доступен на портале госуслуг.

**Что такое штатное расписание и зачем оно?**
Это документ со списком всех должностей в организации, количеством единиц и окладами. Обязателен по закону. Без штатной единицы нельзя принять сотрудника на эту должность.

**Нужно ли вести трудовую книжку в системе?**
Система хранит данные трудовой книжки для справки. Официальная электронная трудовая ведётся в ПФР (Пенсионный фонд). Данные в системе — для удобства HR-отдела.

**Как рассчитываются суточные при командировке?**
Суточные × количество дней. Стандартные суммы: 700 ₽/день по РФ, 2 500 ₽/день за границей (необлагаемый минимум). Точные суммы устанавливает организация.
