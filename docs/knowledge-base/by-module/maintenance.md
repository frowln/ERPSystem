# Техническое обслуживание оборудования (ТОиР)

---

## Что это и зачем

**ТОиР** (техническое обслуживание и ремонт) — модуль для управления стационарным оборудованием на объектах: заявки на ремонт, плановое обслуживание, учёт оборудования.

В отличие от [Автопарка](fleet.md) (транспорт и спецтехника), ТОиР управляет **стационарным** оборудованием: краны, бетононасосы, генераторы, компрессоры, лифты, системы вентиляции.

## Часть 2: Заявки на ремонт

| Поле | Описание |
|------|----------|
| **Оборудование** | Что сломалось |
| **Описание** | Что именно не работает |
| **Приоритет** | Низкий / Средний / Высокий / Критический |
| **Заявитель** | Кто обнаружил поломку |
| **Исполнитель** | Кто будет чинить |
| **Плановая дата** | Когда починят |

### Статусы заявки

| Статус | Описание |
|--------|----------|
| **Новая** | Заявка подана |
| **Принята** | Назначен исполнитель |
| **В работе** | Ремонт идёт |
| **Ожидание запчастей** | Нужна запчасть (заказана) |
| **Выполнена** | Ремонт завершён |
| **Закрыта** | Проверено, оборудование работает |

## Часть 3: Плановое обслуживание

Плановое ТО — это **профилактика**. Лучше потратить 2 часа на замену масла, чем стоять 3 дня из-за сгоревшего мотора.

| Тип ТО | Периодичность | Что делается |
|--------|--------------|-------------|
| **Ежедневное** | Каждый день | Осмотр, смазка, проверка уровней |
| **Еженедельное** | Раз в неделю | Проверка крепежа, фильтров |
| **Ежемесячное** | Раз в месяц | Замена масла, проверка электрики |
| **Сезонное** | 2 раза в год | Подготовка к зиме/лету |
| **Капитальное** | По наработке | Полный ремонт узлов |

Система напоминает о приближении планового ТО и создаёт заявку автоматически.
