# СИЗ (Средства индивидуальной защиты)

---

## Что это и зачем

СИЗ — это средства индивидуальной защиты: каска, перчатки, защитные очки, респиратор, страховочный пояс, сигнальный жилет и другие средства, которые защищают работника от травм и вредных факторов.

По ТК РФ (ст. 221) работодатель **обязан бесплатно** обеспечить работников СИЗ. При проверке инспектор труда спросит: «Покажите ведомость выдачи СИЗ». Нет ведомости — **штраф до 130 000 ₽** за каждого работника.

Модуль «Управление СИЗ» в системе ПРИВОД ведёт учёт:
- Какие СИЗ есть на складе (инвентарь)
- Кому, когда и что выдано
- Когда нужно заменить (по сроку годности или износу)
- Сколько осталось на складе (минимальный запас)

## Часть 1: Инвентарь СИЗ

Инвентарь — это список всех средств защиты, которые есть в наличии у организации.

### Категории СИЗ

| Категория | Что входит | Примеры |
|-----------|-----------|---------|
| **Защита головы** | Каски, подшлемники | Каска строительная СОМЗ-55 |
| **Защита глаз** | Очки, щитки | Очки защитные ЗН8, щиток сварочный |
| **Защита слуха** | Беруши, наушники | Наушники 3M Peltor |
| **Защита органов дыхания** | Респираторы, противогазы | Респиратор 3M 8101, полумаска |
| **Защита рук** | Перчатки, рукавицы | Перчатки х/б, краги сварщика |
| **Защита ног** | Ботинки, сапоги | Ботинки с мет. подноском, сапоги резиновые |
| **Защита тела** | Костюмы, жилеты, фартуки | Костюм «Мастер», фартук прорезиненный |
| **Защита от падения** | Страховочные привязи, стропы | Привязь ПС-1, строп с амортизатором |
| **Сигнальная одежда** | Жилеты, костюмы повышенной видимости | Жилет сигнальный ЖС-01 |
| **Прочее** | Всё остальное | Монтажный пояс, налокотники |

### Поля карточки СИЗ (инвентарь)

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Наименование** | Да | Полное название (до 500 символов) |
| **Артикул (SKU)** | Нет | Код товара |
| **Категория** | Да | Из списка выше (10 категорий) |
| **Размер** | Нет | Размер (S/M/L/XL или числовой) |
| **Сертификат / ГОСТ** | Нет | Ссылка на стандарт (ГОСТ, ТУ) |
| **Общее количество** | Нет | Сколько единиц закуплено |
| **Доступно** | Авто | Сколько на складе (общее − выданное) |
| **Минимальный запас** | Нет | При каком количестве нужно дозакупить |
| **Срок годности** | Нет | До какой даты годен |
| **Примечание** | Нет | Дополнительная информация |

### Индикаторы

- **Зелёный** — достаточный запас (доступно > минимум)
- **Жёлтый** — запас заканчивается (доступно ≤ минимум)
- **Красный** — нет в наличии (доступно = 0)
- **Серый** — срок годности истёк

## Часть 2: Выдача СИЗ

Когда работнику нужны средства защиты — оформляется выдача.

### Как выдать СИЗ

1. На странице **Управление СИЗ** перейдите на вкладку **«Выдача»**
2. Нажмите **«Выдать СИЗ»**
3. Заполните:

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Сотрудник** | Да | Кому выдаём (выбор из списка) |
| **Наименование СИЗ** | Да | Что выдаём (выбор из инвентаря) |
| **Количество** | Да | Сколько единиц (минимум 1) |
| **Дата выдачи** | Да | Когда выдано |
| **Примечание** | Нет | Дополнительная информация |

4. Нажмите **«Выдать»**
5. Система автоматически уменьшает «Доступно» в инвентаре

### Что видно в журнале выдачи

| Колонка | Что показывает |
|---------|---------------|
| Наименование СИЗ | Что выдано |
| Сотрудник | Кому выдано |
| Количество | Сколько единиц |
| Дата выдачи | Когда |
| Дата возврата | Когда вернул (если вернул) |
| Состояние | Статус возврата |

---

## Часть 3: Возврат СИЗ

Когда работник возвращает СИЗ (увольнение, замена, окончание работ):

1. Найдите запись выдачи в журнале
2. Нажмите **«Вернуть»**
3. Укажите **состояние при возврате**:

| Состояние | Описание | Что происходит |
|-----------|---------|---------------|
| **Новое** | Не использовалось | Возвращается в инвентарь |
| **Хорошее** | Можно использовать повторно | Возвращается в инвентарь |
| **Изношенное** | Требует замены, но ещё пригодно | Возвращается, помечается |
| **Повреждённое** | Не пригодно к использованию | Списывается |
| **Списано** | Полностью непригодно | Списывается, не возвращается в инвентарь |

4. При возврате в состоянии «Новое» или «Хорошее» — количество «Доступно» увеличивается обратно
5. При возврате «Повреждённое» или «Списано» — количество не возвращается (СИЗ списан)

## Нормы выдачи СИЗ

По законодательству РФ (Приказ Минтруда) для каждой профессии установлены нормы выдачи:

### Пример: Каменщик

| СИЗ | Количество | Срок носки |
|-----|-----------|-----------|
| Костюм для защиты от общих загрязнений | 1 | 12 месяцев |
| Ботинки кожаные с защитным подноском | 1 пара | 12 месяцев |
| Рукавицы брезентовые | 12 пар | 12 месяцев |
| Каска защитная | 1 | 24 месяца |
| Очки защитные | 1 | до износа |
| Жилет сигнальный | 1 | 12 месяцев |

### Пример: Электросварщик

| СИЗ | Количество | Срок носки |
|-----|-----------|-----------|
| Костюм сварщика | 1 | 12 месяцев |
| Ботинки кожаные с защитным подноском | 1 пара | 12 месяцев |
| Краги сварщика | 6 пар | 12 месяцев |
| Щиток защитный лицевой | 1 | 24 месяца |
| Очки защитные | 1 | до износа |
| Респиратор | 12 шт | 12 месяцев |

> Нормы можно посмотреть в Типовых нормах бесплатной выдачи СИЗ (Приказ Минтруда № 767н).

## Типичные сценарии

### Сценарий 1: Выдача СИЗ новому сотруднику

```
1. Иванов И.И. устроился каменщиком

2. Кладовщик открывает «Управление СИЗ» → «Выдать»:
   → Сотрудник: Иванов И.И.
   → Каска защитная СОМЗ-55 — 1 шт. → Выдано ✅
   → Ботинки с подноском — 1 пара → Выдано ✅
   → Рукавицы — 6 пар → Выдано ✅
   → Жилет сигнальный — 1 шт. → Выдано ✅

3. Инвентарь обновился:
   → Каска: было 50, стало 49
   → Ботинки (р.43): было 12, стало 11
   → Рукавицы: было 120, стало 114
   → Жилеты: было 30, стало 29

4. В карточке Иванова видно: 4 выдачи СИЗ с датами
```

### Сценарий 2: Возврат при увольнении

```
1. Петров П.П. увольняется

2. Кладовщик открывает журнал выдачи → фильтр «Петров»:
   → Каска — выдана 01.06.2025 → Нажимает «Вернуть»
     → Состояние: Изношенное (год носки) → Списано
   → Ботинки — выдана 01.06.2025 → «Вернуть»
     → Состояние: Хорошее → Возвращается в инвентарь ✅
   → Жилет — выдана 01.06.2025 → «Вернуть»
     → Состояние: Хорошее → Возвращается в инвентарь ✅
```

### Сценарий 3: Минимальный запас и дозакупка

```
1. На складе осталось 5 касок (минимум: 10)
   → Система показывает жёлтый индикатор: «Запас ниже минимума»

2. Ответственный видит предупреждение на странице инвентаря

3. Создаёт заявку на закупку (модуль «Закупки»):
   → Каска СОМЗ-55, 30 штук
   → Приоритет: Высокий
   → Обоснование: «Запас СИЗ ниже минимума»

4. После поставки — обновляет инвентарь:
   → Каски: было 5, стало 35
   → Индикатор: зелёный ✅
```

### Сценарий 4: Проверка при инспекции

```
1. Инспектор ГИТ запрашивает ведомость выдачи СИЗ

2. Инженер по ОТ открывает «Управление СИЗ» → вкладка «Выдача»:
   → Фильтр по сотруднику или по проекту
   → Видно: кому, что, когда выдано
   → Можно выгрузить для печати

3. Инспектор проверяет:
   → У всех работников есть записи о выдаче ✅
   → Сроки годности не истекли ✅
   → СИЗ соответствуют профессии ✅
```

## Частые вопросы

**Обязан ли работодатель выдавать СИЗ бесплатно?**
Да. По ТК РФ (ст. 221) работодатель обязан за свой счёт обеспечить работников СИЗ, прошедшими обязательную сертификацию.

**Что делать, если работник отказывается носить СИЗ?**
Работодатель обязан не допускать к работе без СИЗ. Отказ — основание для отстранения от работы без оплаты (ст. 76 ТК РФ).

**Как часто нужно менять СИЗ?**
По нормам выдачи — обычно раз в 12 месяцев для одежды и обуви, раз в 24 месяца для касок. Респираторы — чаще. При повреждении — немедленно.

**Кто отвечает за учёт СИЗ?**
Обычно кладовщик или инженер по ОТ. В системе ПРИВОД учёт ведётся автоматически при выдаче и возврате.

**Нужно ли вести бумажный журнал выдачи?**
Формально требуется ведомость выдачи (по форме). Система ПРИВОД ведёт электронный учёт, который можно распечатать для проверки.

**Что если СИЗ нет на складе?**
Система показывает предупреждение. Работника нельзя допускать к работе без необходимых СИЗ. Нужно срочно закупить.

**Можно ли выдать одни СИЗ нескольким работникам?**
Одноразовые СИЗ (респираторы, перчатки) — нет, каждому свои. Многоразовые (каска) — можно, если вернули в хорошем состоянии.

**Где в системе посмотреть, что выдано конкретному сотруднику?**
На странице «Управление СИЗ» → вкладка «Выдача» → фильтр по сотруднику. Или в карточке сотрудника.
