# Исполнительная документация (ПТО)

---

## Что это и зачем

**Исполнительная документация** (ИД) — это комплект документов, которые подтверждают, что строительные работы выполнены в соответствии с проектом и нормативами. Без исполнительной документации объект **не примут в эксплуатацию**.

По СП 48.13330 (Организация строительства) застройщик обязан вести исполнительную документацию на протяжении всего строительства.

В модуле ПТО (производственно-технический отдел) системы ПРИВОД ведутся:
- **АОСР** — акты освидетельствования скрытых работ
- **Общий журнал работ (КС-6)** — ежедневный журнал хода строительства
- **Журнал входного контроля** — приёмка материалов
- **Журнал сварочных работ** — учёт сварных соединений
- **Специальные журналы** — бетонные, монтажные, свайные работы
- **Наряд-допуски** — разрешения на опасные работы
- **Лабораторные испытания** — протоколы проверки материалов

## Часть 1: АОСР (Акт освидетельствования скрытых работ)

**АОСР** — один из главных документов исполнительной документации. Он подтверждает, что работы, которые будут скрыты последующими конструкциями (арматура перед бетонированием, гидроизоляция перед засыпкой), выполнены правильно.

**Почему это важно:** После бетонирования вы уже не увидите арматуру. Единственное подтверждение, что она уложена по проекту — подписанный АОСР.

### Поля АОСР

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Номер акта** | Да | Порядковый номер (например: АОСР-001) |
| **Проект** | Да | К какому объекту |
| **Дата осмотра** | Да | Когда проводился осмотр |
| **Описание работ** | Да | Какие скрытые работы освидетельствуются |
| **Место** | Да | Где (оси, отметки, этаж) |
| **Подрядчик** | Да | Кто выполнял работы |
| **Инспектор** | Да | Кто осматривал |
| **Фото** | Рекомендуется | Фотофиксация до закрытия |
| **Примечание** | Нет | Дополнительная информация |

### Статусы АОСР

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Черновик** (DRAFT) | Серый | Акт создан, подписи не собраны |
| **Ожидает осмотра** (PENDING_INSPECTION) | Синий | Назначен осмотр, ждём инспектора |
| **Осмотрен** (INSPECTED) | Жёлтый | Осмотр проведён, идёт сбор подписей |
| **Утверждён** (APPROVED) | Зелёный | Все подписи собраны, акт закрыт |
| **Отклонён** (REJECTED) | Красный | Работы не соответствуют — переделка |

```
Черновик → Ожидает осмотра → Осмотрен → Утверждён
                                       → Отклонён
```

### Подписи на АОСР

АОСР подписывают представители всех сторон:

| Роль подписанта | Кто это |
|----------------|---------|
| **Представитель застройщика** | Инженер технического надзора заказчика |
| **Представитель подрядчика** | Прораб или начальник участка |
| **Авторский надзор** | Представитель проектной организации |
| **Технический надзор** | Инженер строительного контроля |
| **Иной инспектор** | Представитель Ростехнадзора (при необходимости) |

Каждая подпись имеет статус:
- **Ожидает подписания** — подпись ещё не поставлена
- **Подписано** — человек подписал (можно подписать [КЭП](digital-signature.md))
- **Отклонено** — подписант отклонил (с указанием причины)

### Вложения к АОСР

К акту прикрепляются документы:

| Тип вложения | Что это |
|-------------|---------|
| **Фото** | Фотофиксация скрытых работ |
| **Геодезическая съёмка** | Исполнительная геодезия |
| **Сертификат материала** | Паспорт/сертификат на использованный материал |
| **Чертёж** | Проектный чертёж или исполнительная схема |
| **Исполнительная схема** | Фактическое расположение конструкций |
| **Прочее** | Любые другие документы |

## Часть 2: Общий журнал работ (КС-6)

**Общий журнал работ** — обязательный документ на каждом строительном объекте. В него записывается всё, что происходит на стройке каждый день.

**Путь:** Исполнительная документация → **Журнал КС-6**

### Что записывается в журнал

| Поле | Описание |
|------|----------|
| **Дата** | Дата записи |
| **Описание работ** | Какие работы выполнялись |
| **Единица измерения** | м³, м², шт, т и т.д. |
| **Количество** | Объём выполненных работ |
| **Подрядчик** | Кто выполнял |
| **Погода** | Погодные условия (температура, осадки) |
| **Численность** | Сколько рабочих на площадке |
| **Техника** | Какая техника использовалась |
| **Примечание** | Дополнительная информация |

### Статусы журнала

| Статус | Описание |
|--------|----------|
| **Черновик** | Журнал создан, заполняется |
| **Активный** | Журнал ведётся ежедневно |
| **Закрыт** | Журнал закрыт (объект сдан или этап завершён) |
| **Архивирован** | Перемещён в архив |

Система предоставляет **календарный вид** — удобно видеть, по каким дням есть записи, а по каким пропущены.

## Часть 3: Журнал входного контроля

Каждый материал, который приходит на стройку, должен быть проверен и записан в журнал входного контроля.

**Путь:** Исполнительная документация → **Входной контроль**

### Что записывается

| Поле | Описание |
|------|----------|
| **Дата** | Дата приёмки |
| **Материал** | Название материала |
| **Поставщик** | От кого получен |
| **Номер документа** | Номер накладной или сертификата |
| **Количество** | Сколько |
| **Единица** | Тонны, м³, штуки и т.д. |
| **Инспектор** | Кто принимал |
| **Результат** | Принято / Отклонено / Условно |

## Часть 4: Журнал сварочных работ

Ведётся на объектах, где выполняются сварные соединения (металлоконструкции, трубопроводы).

**Путь:** Исполнительная документация → **Сварочные работы**

### Что записывается

| Поле | Описание |
|------|----------|
| **Дата** | Дата сварки |
| **Номер шва** | Уникальный номер сварного соединения |
| **Сварщик** | ФИО сварщика |
| **Удостоверение** | Номер аттестации НАКС |
| **Тип соединения** | Стыковое, угловое, тавровое |
| **Материал** | Марка стали |
| **Электрод** | Марка электрода/проволоки |
| **Метод контроля** | Визуальный, УЗК, радиография |
| **Результат** | Годен / Брак |

## Часть 5: Специальные журналы

Для отдельных видов работ ведутся специальные журналы:

| Журнал | Когда ведётся |
|--------|--------------|
| **Журнал бетонных работ** | При бетонировании конструкций |
| **Журнал монтажных работ** | При монтаже металлоконструкций, сборных ЖБ |
| **Журнал свайных работ** | При устройстве свайных оснований |

В каждом журнале фиксируются: дата, описание работ, объём, погода, ответственный.

## Часть 6: Наряд-допуски

**Наряд-допуск** — это письменное разрешение на выполнение опасных работ. Без наряда-допуска начинать такие работы **запрещено**.

**Путь:** ПТО → **Наряд-допуски**

### Типы работ, требующие наряд-допуск

| Тип работ | Примеры |
|-----------|---------|
| **Бетонные работы** | Заливка фундамента, перекрытий |
| **Арматурные работы** | Вязка каркасов |
| **Фундаментные работы** | Устройство оснований |
| **Гидроизоляция** | Обмазочная, оклеечная |
| **Теплоизоляция** | Утепление фасадов |
| **Электромонтажные** | Прокладка кабелей, монтаж щитов |
| **Сантехнические** | Монтаж трубопроводов |
| **Вентиляция** | Монтаж воздуховодов |
| **Сварочные** | Огневые работы |
| **Земляные** | Рытьё котлованов |
| **Отделочные** | Штукатурка, покраска |
| **Кровельные** | Работы на высоте |
| **Другое** | Прочие опасные работы |

### Статусы наряда-допуска

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Черновик** (DRAFT) | Серый | Создан, не выдан |
| **Выдан** (ISSUED) | Синий | Наряд оформлен и выдан ответственному |
| **Действует** (ACTIVE) | Зелёный | Работы ведутся по наряду |
| **Приостановлен** (SUSPENDED) | Жёлтый | Работы временно остановлены (погода, авария) |
| **Закрыт** (CLOSED) | Серый | Работы завершены, наряд закрыт |
| **Истёк** (EXPIRED) | Красный | Срок действия наряда истёк |

```
Черновик → Выдан → Действует → Закрыт
                       ↓           ↑
                  Приостановлен ──→'
                       ↓
                    Истёк
```

### Поля наряда-допуска

| Поле | Описание |
|------|----------|
| **Номер** | Уникальный код (автоматически) |
| **Тип работ** | Из списка выше (13 типов) |
| **Место** | Где будут выполняться работы |
| **Дата начала** | Когда начинаются работы |
| **Дата окончания** | Когда заканчиваются |
| **Кто выдал** | Ответственное лицо |
| **Кто утвердил** | Утверждающий |
| **Меры безопасности** | Что нужно обеспечить (ограждения, СИЗ и т.д.) |

## Часть 7: Лабораторные испытания

Строительные материалы проходят лабораторные испытания для подтверждения качества.

**Путь:** ПТО → **Лабораторные**

### Типы испытаний

| Тип | Что проверяется | Пример |
|-----|----------------|--------|
| **Прочность** | Прочность на сжатие/растяжение | Бетонные кубики на 7-й и 28-й день |
| **Плотность** | Плотность материала | Плотность грунта после уплотнения |
| **Влажность** | Содержание воды | Влажность древесины |
| **Уплотнение** | Степень уплотнения | Коэффициент уплотнения грунта |
| **Морозостойкость** | Устойчивость к замораживанию | Циклы замораживания бетона |
| **Водонепроницаемость** | Непроницаемость для воды | Марка водонепроницаемости W8 |
| **Растяжение** | Прочность на разрыв | Испытание арматурных стержней |
| **Химический анализ** | Состав материала | Анализ воды для бетона |
| **Другое** | Прочие испытания | Теплопроводность, огнестойкость |

### Заключение по испытанию

| Заключение | Цвет | Что означает |
|------------|------|-------------|
| **Соответствует** (PASSED) | Зелёный | Материал прошёл испытание |
| **Не соответствует** (FAILED) | Красный | Материал не прошёл — использовать нельзя |
| **Условно годен** (CONDITIONAL) | Жёлтый | Есть отклонения, но можно использовать с ограничениями |
| **Ожидает результатов** (PENDING) | Серый | Испытание проведено, результаты обрабатываются |

| **Журнал работ** | КС-6, спец. журналы |
| **Проектная документация** | Чертежи, расчёты |
| **Прочее** | Любые другие документы |

### Дисциплины

| Дисциплина | Обозначение |
|------------|-------------|
| Архитектурный | АР |
| Конструктивный | КР |

| Электрика | ЭО |
| Водоснабжение и канализация | ВК |
| Вентиляция | ОВ |
| Пожарная безопасность | ПБ |
| Генплан | ГП |
| Ландшафт | ЛА |

### Статусы документа ПТО

| Статус | Цвет | Описание |
|--------|------|----------|
| **Черновик** | Серый | Создан, не отправлен на проверку |
| **На проверке** | Синий | Отправлен на проверку ответственному |
| **Утверждён** | Зелёный | Проверен и утверждён |
| **Отклонён** | Красный | Возвращён с замечаниями |
| **В архиве** | Серый | Перемещён в архив |

```
Черновик → На проверке → Утверждён → В архиве
                       → Отклонён → Черновик (исправление)
```

## Типичные сценарии

### Сценарий 1: Оформление АОСР на армирование

```
1. Арматурщики завершили армирование плиты 2 этажа

2. Прораб создаёт АОСР:
   → Номер: АОСР-045
   → Работы: «Армирование монолитной плиты перекрытия 2 этажа»
   → Место: Оси А-Г / 1-6, отметка +3.300
   → Фото: 8 фотографий с разных ракурсов
   → Вложения: исполнительная схема армирования

3. Сбор подписей:
   → Представитель подрядчика (прораб) → Подписано ✅
   → Авторский надзор (проектировщик) → Подписано ✅
   → Технический надзор (заказчик) → Подписано ✅
   → Все подписи можно поставить КЭП (электронной подписью)

4. Статус: Утверждён ✅
   → Можно бетонировать

5. Без АОСР бетонирование начинать НЕЛЬЗЯ
```

### Сценарий 2: Ежедневная запись в журнал КС-6

```
1. Каждый день в 17:00 начальник участка заполняет журнал:

   05.03.2026:
   → Бетонирование плиты 2 этажа, В25, V=45 м³
   → Погода: -5°С, ясно
   → Численность: 18 человек
   → Техника: автобетононасос, вибратор
   → Примечание: обогрев бетона электрокабелем (зимнее бетонирование)

2. В календарном виде видно: зелёные дни (есть запись), красные (нет записи)

3. При сдаче объекта журнал предъявляется приёмочной комиссии
```

### Сценарий 3: Лабораторное испытание бетона

```
1. 01.03 — залита плита 2 этажа, бетон В25

2. Лаборант отбирает 3 серии кубиков (по 3 штуки):
   → Серия 1: испытание на 7-й день (08.03)
   → Серия 2: испытание на 28-й день (29.03)
   → Серия 3: контрольная (хранение)

3. 08.03 — испытание на 7-й день:
   → Тип: Прочность
   → Результат: 22.4 МПа (68% от R28 = 32.7 МПа)
   → Заключение: Соответствует (норма: ≥ 50% на 7-й день) ✅

4. 29.03 — испытание на 28-й день:
   → Результат: 34.1 МПа (104% от проектной)
   → Заключение: Соответствует ✅
   → Протокол прикреплён к карточке

5. Протокол испытания прикладывается к АОСР
   и входит в комплект исполнительной документации
```

## Частые вопросы

**Обязательна ли исполнительная документация?**
Да. По СП 48.13330 и Градостроительному кодексу ведение исполнительной документации обязательно. Без неё объект не примут в эксплуатацию.

**Кто ведёт исполнительную документацию?**
Отдел ПТО (производственно-технический отдел) подрядной организации. Контроль — технический надзор заказчика.

**Что будет, если не оформить АОСР?**
При сдаче объекта приёмочная комиссия потребует все АОСР. Если акта нет — может потребоваться вскрытие конструкции для проверки скрытых работ. Это дорого и долго.

**Можно ли подписать АОСР электронной подписью?**
Да. Система ПРИВОД поддерживает подписание [КЭП](digital-signature.md) (квалифицированной электронной подписью). Это юридически равнозначно собственноручной подписи.

**Как часто нужно заполнять журнал КС-6?**
Ежедневно, в каждый рабочий день. Пропуски — нарушение. Система показывает в календарном виде, какие дни заполнены, а какие — нет.

**Зачем хранить протоколы лабораторных испытаний?**
Это доказательство, что материалы соответствуют проекту. При любом споре (трещина в стене через 5 лет) протокол покажет, какой бетон использовали и какой прочности он достиг.

**Что такое наряд-допуск простыми словами?**
Это «разрешение на работу». Нельзя начать опасные работы (сварку, работу в котловане, на высоте) без письменного разрешения с указанием мер безопасности.
