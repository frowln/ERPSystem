# База расценок (Ценообразование)

---

## Что это и зачем

База расценок — это справочник нормативных цен на строительные работы, материалы и механизмы. Здесь хранятся расценки ГЭСН, ФЕР, ТЕР — те самые нормативы, по которым считаются сметы. Система позволяет искать расценки, смотреть их структуру (сколько стоит работа, материалы, механизмы) и пересчитывать в текущие цены с помощью индексов Минстроя.

Это рабочий инструмент сметчика: вместо того чтобы листать бумажные сборники, можно за секунду найти нужную расценку и рассчитать стоимость с учётом региона и квартала.

---

## Где найти

**Путь:** Главное меню → **Сметы** → **Базы ценообразования**

Также доступны:
- **Расценки:** Сметы → Расценки
- **Калькулятор цен:** Сметы → Калькулятор цен

---

## Базы ценообразования

Каждая база — это сборник расценок определённого типа.

**Типы баз:**

| Тип | Цвет метки | Что содержит |
|-----|-----------|-------------|
| **ФЕР** | Синий | Федеральные единичные расценки — общероссийские цены на работы |
| **ТЕР** | Зелёный | Территориальные единичные расценки — региональные цены (точнее, т.к. учитывают местные условия) |
| **ГЭСН** | Фиолетовый | Государственные элементные сметные нормы — расход ресурсов на единицу работы |
| **Локальная** | Жёлтый | Собственная база компании — ваши внутренние расценки |

**Данные базы:**
- Название
- Тип (ФЕР/ТЕР/ГЭСН/Локальная)
- Регион (для ТЕР — например, «Москва», «Свердловская область»)
- Базовый год (год, к которому привязаны цены — например, 2020)
- Коэффициент пересчёта (множитель для получения текущих цен)
- Период действия (дата начала и окончания)
- Статус (активная / неактивная)

**Фильтры:**
- Поиск по названию, региону или типу
- Вкладки: Все / Активные / Неактивные

## Расценки

**Путь:** Сметы → Расценки

Каждая расценка — это одна работа или материал с разбивкой по стоимости.

**Структура расценки:**

| Составляющая | Что значит | Пример |
|-------------|-----------|--------|
| **Код** | Номер расценки в сборнике | ФЕР 11-01-001 |
| **Наименование** | Описание работы | «Кладка стен из кирпича» |
| **Единица** | В чём измеряется | м3, м2, шт, 100 м |
| **Оплата труда** | Стоимость рабочей силы на единицу | 450 руб./м3 |
| **Материалы** | Стоимость материалов на единицу | 1 200 руб./м3 |
| **Механизмы** | Стоимость машин/оборудования на единицу | 180 руб./м3 |
| **Накладные** | Накладные расходы на единицу | 320 руб./м3 |
| **Итого** | Сумма всех составляющих | 2 150 руб./м3 |

**Поиск расценки:**
- Введите код (например, «11-01-001») или название (например, «кладка кирпича»)
- Выберите базу из выпадающего списка (или ищите по всем базам)
- Результаты появятся автоматически (с задержкой 0.4 сек для поиска по серверу)

**Импорт/Экспорт:**
- **Импорт из CSV** — загрузить расценки из файла (только для администраторов)
- **Экспорт в CSV** — скачать расценки выбранной базы

## Калькулятор цен

**Путь:** Сметы → Калькулятор цен

Калькулятор позволяет быстро рассчитать стоимость работы с учётом объёма и текущих индексов.

**Как пользоваться:**

1. **Найдите расценку** — введите код или название в строку поиска. Выберите нужную из списка
2. **Укажите параметры:**
   - **Количество** — сколько единиц (например, 150 м3)
   - **Регион** — для подбора правильного индекса пересчёта
3. **Посмотрите результат:**

| Строка | Что показывает |
|--------|---------------|
| Базовая стоимость | Расценка x Количество (в ценах базового года) |
| Индекс пересчёта | Коэффициент Минстроя для вашего региона и квартала (например, 9.45) |
| Цена за единицу (текущая) | Расценка x Индекс |
| **Итого (текущая)** | Цена за единицу x Количество |

Также показывается разбивка по составляющим:

| Составляющая | За единицу | Итого |
|-------------|-----------|-------|
| Оплата труда | 450 | 67 500 |
| Материалы | 1 200 | 180 000 |
| Механизмы | 180 | 27 000 |
| Накладные | 320 | 48 000 |
| **ИТОГО** | 2 150 | 322 500 |

> **Пример:** Расценка «Кладка кирпича» = 2 150 руб./м3 (базовые цены). Индекс для Москвы, Q1 2026 = 9.45. Объём = 150 м3. Итого текущая стоимость = 2 150 x 9.45 x 150 = **3 047 625 руб.**

## Индексы Минстроя

Индексы — это квартальные коэффициенты пересчёта, публикуемые Минстроем РФ. Они переводят базовые цены (2001 года или другого базового периода) в текущие рыночные цены.

**Где посмотреть:** Вкладка «Индексы» в калькуляторе цен (или Сметы → Индексы Минстроя)

**Колонки таблицы индексов:**
- Регион
- Вид работ
- Базовый квартал (например, Q1 2020)
- Целевой квартал (например, Q2 2026)
- Значение индекса (например, 1.2456)
- Источник

**Фильтры:**
- Поиск по региону
- Поиск по виду работ

## Ценовые коэффициенты

**Путь:** Главное меню → **Ценовые коэффициенты**

Коэффициенты — это множители, которые применяются к базовой цене для учёта различных условий.

**Типы коэффициентов:**

| Тип | Пример | Значение |
|-----|--------|---------|
| **Региональный** | Работа на Крайнем Севере | 1.15 (удорожание 15%) |
| **Сезонный** | Зимнее строительство | 1.10 (зимнее удорожание) |
| **Материальный** | Дефицитный материал | 1.20 |
| **Трудовой** | Высокая квалификация рабочих | 1.08 |
| **Оборудование** | Редкая техника | 1.12 |
| **Накладные** | Повышенные условия | 1.05 |
| **Произвольный** | Индивидуальная надбавка | любое значение |

**Данные коэффициента:**
- Название и описание
- Тип (региональный, сезонный и т.д.)
- Значение (множитель, например 1.15)
- Статус (активный / неактивный / черновик / истёк)
- Проект и/или договор (привязка)
- Период действия (дата начала / окончания)

**Как применить:**
Коэффициенты применяются автоматически при расчёте стоимости через API. Можно также рассчитать скорректированную цену вручную: выберите базовую цену и набор коэффициентов — система перемножит.

## Частые вопросы

**Чем ФЕР отличается от ТЕР?**
ФЕР — федеральные расценки, одинаковые для всей России. ТЕР — территориальные, для конкретного региона. ТЕР точнее, потому что учитывают местные цены на материалы и рабочую силу. Если есть ТЕР для вашего региона — используйте их.

**Что такое ГЭСН?**
Это нормы расхода ресурсов: сколько нужно рабочих, материалов и механизмов на единицу работы. ГЭСН — основа для расчёта ФЕР и ТЕР. Если вы сметчик и составляете смету ресурсным методом — вам нужны ГЭСН.

**Зачем нужен индекс пересчёта?**
Все нормативные расценки (ФЕР/ТЕР) указаны в базовых ценах (часто 2001 года). С тех пор цены выросли в разы. Индекс — это множитель, который переводит старые цены в актуальные. Без индекса ваша смета будет в ценах 20-летней давности.

**Как часто обновляются индексы?**
Минстрой публикует индексы ежеквартально. Рекомендуется обновлять индексы в системе сразу после публикации нового письма Минстроя.

**Можно ли создать свою базу расценок?**
Да. Выберите тип «Локальная» и добавляйте свои расценки вручную или импортом из CSV. Это удобно для внутренних расценок компании, которые отличаются от нормативных.

**Как применить зимнее удорожание?**
Создайте ценовой коэффициент типа «Сезонный» со значением 1.10–1.15 (в зависимости от региона и вида работ). Привяжите к проекту или договору. При расчёте стоимости коэффициент применится автоматически.

**Как найти расценку по коду?**
Перейдите в Сметы → Расценки. В строку поиска введите код (например, «ФЕР 11-01-001» или просто «11-01-001»). Результаты появятся автоматически.
