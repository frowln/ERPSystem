# AI-ассистент

---

## Что это и зачем

**AI-ассистент** — встроенный помощник на базе искусственного интеллекта. Отвечает на вопросы, помогает найти информацию в системе, подсказывает действия.

Не нужно звонить в поддержку или искать в документации — спросите ассистента прямо в системе.

---

## Где найти

**Путь:** Иконка чата в правом нижнем углу экрана (на любой странице)

---

## Что умеет

| Возможность | Пример |
|------------|--------|
| **Ответы на вопросы** | «Как создать КС-2?» → пошаговая инструкция |
| **Поиск данных** | «Покажи все просроченные задачи по проекту ЖК Южный» |
| **Навигация** | «Где настроить уведомления?» → ссылка на нужную страницу |
| **Расчёты** | «Сколько бетона нужно на плиту 12×8×0.3 м?» |
| **Объяснение терминов** | «Что такое CPI?» → индекс стоимости, формула, пример |
| **Помощь с ошибками** | «Почему не могу создать проект?» → проверка квоты и прав |

---

## Как пользоваться

1. Нажмите на иконку чата (правый нижний угол)
2. Напишите вопрос на русском языке
3. Ассистент ответит с учётом контекста (какая страница открыта, какой проект выбран)
4. Можно задавать уточняющие вопросы в диалоге

> **Контекст страницы:** Если вы на странице бюджета проекта «ЖК Южный» и спрашиваете «какой остаток?» — ассистент поймёт, что вы про бюджет этого конкретного проекта.

---

## Ограничения

- Ассистент не может изменять данные — только читать и подсказывать
- Ответы основаны на данных в системе и базе знаний
- Для сложных финансовых вопросов лучше обратиться к бухгалтеру
