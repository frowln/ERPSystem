# Почта

---

## Что это и зачем

**Почта** — это встроенный почтовый клиент внутри системы ПРИВОД. Он позволяет читать, писать и отправлять электронные письма, не выходя из системы.

Главное преимущество — **привязка писем к проектам**. Когда переписка с заказчиком привязана к проекту, любой член команды может найти нужное письмо в контексте проекта, а не искать по всей почте.

Без встроенной почты:
- Важное письмо от заказчика теряется в почтовом ящике менеджера
- Уходит в отпуск — никто не знает, о чём договорились
- Нужно искать вложение к письму — а оно в личной почте сотрудника
- Переписка по проекту разбросана по десяткам цепочек в разных ящиках

Встроенная почта решает:
- Все рабочие письма видны в системе
- Привязка к проектам — вся переписка по объекту в одном месте
- Ответ, пересылка, вложения — всё как в обычной почте
- Синхронизация с внешним почтовым сервером (IMAP)

## Часть 1: Интерфейс почты

Страница почты состоит из **трёх колонок**:

### Левая колонка — Папки

| Папка | Описание |
|-------|----------|
| **Входящие** | Полученные письма (с индикатором непрочитанных) |
| **Отправленные** | Письма, которые вы отправили |
| **Черновики** | Незаконченные письма |
| **Корзина** | Удалённые письма |

Здесь же кнопка **«Написать»** — открывает окно нового письма.

### Средняя колонка — Список писем

- Поиск по теме и отправителю
- Кнопка **«Синхронизировать»** — загрузить новые письма с сервера
- Список писем с пагинацией
- Каждое письмо показывает: отправителя, тему, время, значок вложения, звёздочку

**Непрочитанные** письма выделены жирным шрифтом и синей полоской слева.

### Правая колонка — Чтение письма

Когда вы выбираете письмо из списка, справа отображается:
- **Тема** (крупно)
- **От кого** → **Кому** (и **Копия**, если есть)
- **Дата получения**
- **Кнопки действий**: Ответить / Ответить всем / Переслать / Привязать к объекту
- **Привязанные проекты** (если есть)
- **Вложения** (с возможностью скачать)
- **Текст письма**

## Часть 2: Написание письма

Нажмите **«Написать»** — откроется модальное окно:

| Поле | Описание |
|------|----------|
| **Кому** | Адрес получателя (можно несколько, через запятую или точку с запятой) |
| **Копия** | Адреса для копии (CC) — необязательно |
| **Тема** | О чём письмо |
| **Текст** | Содержание письма |
| **Вложения** | Кнопка «Прикрепить файл» — можно добавить несколько файлов |

Нажмите **«Отправить»** — письмо уходит.

---

## Часть 3: Ответ и пересылка

### Ответить

- Нажмите **«Ответить»** на открытом письме
- В поле «Кому» автоматически подставляется отправитель
- Тема: «Re: ...» (автоматически)
- Можно добавить вложения

### Ответить всем

- Нажмите **«Ответить всем»**
- В «Кому» — отправитель, в «Копия» — все получатели оригинала
- Удобно для групповых обсуждений

### Переслать

- Нажмите **«Переслать»**
- Поле «Кому» пустое — укажите нового получателя
- Тема: «Fwd: ...»
- Оригинальное письмо включается в пересылку

---

## Часть 4: Привязка к проектам

Это **главная особенность** встроенной почты — возможность привязать письмо к проекту.

### Как привязать

1. Откройте письмо
2. Нажмите **«Привязать к объекту»**
3. Найдите проект в поиске
4. Нажмите на проект — письмо привязано

### Что это даёт

- На карточке проекта видна вся привязанная переписка
- Любой член команды проекта может найти нужное письмо
- Не нужно пересылать — достаточно привязать

### Отвязать

Если привязали по ошибке — нажмите ✕ рядом с названием проекта.

---

## Часть 5: Звёздочки и прочитанные

### Звёздочка (⭐)

Нажмите звёздочку, чтобы отметить важное письмо. Звёздочка помогает быстро находить важные письма в потоке.

### Прочитанные / Непрочитанные

- **Непрочитанное** — жирный шрифт, синяя полоска
- **Прочитанное** — обычный шрифт

Письмо автоматически помечается как прочитанное, когда вы его открываете. Можно вернуть статус «непрочитанное», если хотите обработать позже.

---

## Часть 6: Вложения

### Отправка вложений

1. В окне написания нажмите **«Прикрепить файл»**
2. Выберите файл на компьютере
3. Файл загружается (показывается прогресс)
4. Можно удалить вложение до отправки (кнопка ✕)
5. При отправке письма — вложения уходят вместе с ним

### Скачивание вложений

На открытом письме видны все вложения:
- Название файла
- Размер
- Кнопка скачивания

---

## Часть 7: Синхронизация

Кнопка **«Синхронизировать»** загружает новые письма с внешнего почтового сервера (IMAP).

Это нужно, потому что:
- Почта ПРИВОД работает как **почтовый клиент** (аналог Outlook/Thunderbird)
- Она подключается к вашему почтовому серверу и загружает письма
- Новые письма не появляются мгновенно — нужно нажать «Синхронизировать» или подождать автоматическую синхронизацию

## Типичные сценарии

### Сценарий 1: Привязка переписки с заказчиком к проекту

```
1. Менеджер получил письмо от заказчика:
   → От: kozlov@ugstroy.ru
   → Тема: «Замечания к проекту ЖК Южный парк»
   → Вложения: «Замечания_v2.pdf»

2. Открывает письмо → привязывает к проекту:
   → «Привязать к объекту» → ЖК Южный парк ✅

3. Отвечает:
   → «Ответить» → «Добрый день! Замечания приняты,
     устраним до 25.04. С уважением, ...»

4. Через неделю инженер ПТО ищет информацию по проекту:
   → Открывает проект → видит привязанные письма
   → Находит «Замечания_v2.pdf» → скачивает
   → Не нужно спрашивать менеджера!
```

### Сценарий 2: Пересылка письма коллеге

```
1. Снабженец получил счёт от поставщика:
   → Тема: «Счёт №234 от 10.04.2026»
   → Вложение: «Счёт_234.pdf»

2. Пересылает бухгалтеру:
   → «Переслать» → sidorova@privod.ru
   → Добавляет комментарий: «Прошу оплатить до 15.04»

3. Привязывает к проекту: ЖК Южный парк

4. Бухгалтер получает → оплачивает → отвечает:
   → «Оплачено, п/п №567 от 12.04»
```

## Частые вопросы

**Это замена моей обычной почте?**
Нет. Встроенная почта работает **параллельно** с вашей обычной почтой (Gmail, Яндекс, корпоративная). Она синхронизирует письма с вашего почтового сервера и показывает их внутри системы. Вы по-прежнему можете пользоваться обычной почтой.

**Увидят ли коллеги мою личную переписку?**
Нет. Каждый пользователь видит только **свою** почту. Другие пользователи видят письма, только если вы привязали их к общему проекту.

**Что значит «привязать к объекту»?**
Это значит создать связь между письмом и проектом. После этого все участники проекта смогут найти это письмо на странице проекта. Само письмо остаётся в вашем ящике.

**Можно ли привязать одно письмо к нескольким проектам?**
Да. Если письмо касается нескольких проектов — привяжите его ко всем.

**Что будет, если я удалю письмо?**
Оно переместится в «Корзину». Из корзины можно восстановить.

**Как часто обновляется почта?**
При нажатии «Синхронизировать». Также есть периодическая автоматическая синхронизация.

**Поддерживается ли отправка с корпоративного домена?**
Да, если почтовый сервер настроен в системе. Письма уходят от вашего рабочего адреса (например: ivanov@stroykompaniya.ru).
