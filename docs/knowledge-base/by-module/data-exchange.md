# Обмен данными (Импорт / Экспорт)

---

## Что это и зачем

**Обмен данными** — модуль для массового импорта данных в систему и экспорта из неё. Полезен при первоначальной настройке (перенос из Excel), при интеграции с 1С, и для выгрузки отчётов.

---

## Где найти

**Путь:** Главное меню → **Обмен данными**

| Страница | URL | Что делает |
|----------|-----|-----------|
| **Импорт** | /data-exchange/import | Загрузка данных из файлов |
| **Экспорт** | /data-exchange/export | Выгрузка данных в файлы |
| **Настройка 1С** | /data-exchange/1c-config | Конфигурация обмена с 1С |

---

## Часть 1: Импорт

### Поддерживаемые форматы

| Формат | Что можно импортировать |
|--------|------------------------|
| **XLSX** | Контрагенты, сотрудники, материалы, сметы, спецификации |
| **CSV** | Любые табличные данные |
| **XML** | Данные из 1С, ИСУП |

### Процесс импорта

1. Выберите тип данных (контрагенты, материалы, сотрудники...)
2. Скачайте шаблон (Excel с правильными колонками)
3. Заполните шаблон данными
4. Загрузите файл
5. Система покажет предпросмотр — проверьте
6. Подтвердите импорт

> **Совет:** При первоначальной настройке импортируйте в таком порядке: контрагенты → сотрудники → материалы → проекты.

---

## Часть 2: Экспорт

| Что экспортировать | Формат | Назначение |
|-------------------|--------|-----------|
| **Отчёты** | XLSX, PDF | Для руководства, заказчиков |
| **Реестры** | XLSX, CSV | Для анализа в Excel |
| **Документы** | PDF | Для архива, печати |
| **Данные для 1С** | XML | Для бухгалтерии |

---

## Часть 3: Обмен с 1С

Двусторонний обмен с 1С:Бухгалтерия:

| Направление | Данные |
|-------------|--------|
| **ПРИВОД → 1С** | КС-2/КС-3, счета, акты, контрагенты |
| **1С → ПРИВОД** | Платежи, банковская выписка, остатки по счетам |

→ Подробнее: [Интеграции](integrations.md)
