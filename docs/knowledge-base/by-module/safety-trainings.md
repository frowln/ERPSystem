# Обучение и инструктажи по охране труда

---

## Что это и зачем

Обучение и инструктажи — это обязательные мероприятия по безопасности для всех работников. Без прохождения инструктажа **работник не имеет права выходить на площадку**. Если произойдёт несчастный случай, а работник не проходил инструктаж — **уголовная ответственность** для руководителя.

В системе ПРИВОД два связанных раздела:
- **Обучение** — групповые мероприятия (курсы, семинары, тренинги)
- **Инструктажи** — персональные или групповые инструктажи с подписями участников

## Часть 1: Обучение

### Виды обучения

| Тип | Описание | Когда проводится |
|-----|---------|-----------------|
| **Вводный** | Общее знакомство с правилами безопасности организации | При приёме на работу |
| **Периодический** | Повторение и обновление знаний | По графику (ежегодно, раз в 3 года) |
| **Внеплановый** | После инцидента, изменения правил, нового оборудования | По необходимости |
| **Целевой** | Для конкретной опасной работы или задачи | Перед началом работ |

**Цвета типов:**
- Вводный — синий
- Периодический — жёлтый
- Внеплановый — оранжевый
- Целевой — красный

### Как создать обучение

1. Перейдите **Безопасность → Обучение**
2. Нажмите **«+ Новое обучение»**

3. Заполните:

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Название** | Да | Тема обучения (до 500 символов). Например: «Работы на высоте: правила и СИЗ» |
| **Тип** | Да | Вводный / Периодический / Внеплановый / Целевой |
| **Дата** | Да | Когда проводится |
| **Проект** | Нет | На каком объекте (если привязано к проекту) |
| **Инструктор** | Нет | Кто проводит обучение (выбор из сотрудников) |
| **Участники** | Нет | Список участников (до 5000 символов) |
| **Темы** | Нет | Программа обучения (до 2000 символов) |
| **Длительность** | Нет | В минутах |
| **Примечание** | Нет | Дополнительная информация |

4. Нажмите **«Создать»** → обучение в статусе «Запланировано»

### Статусы обучения

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Запланировано** | Синий | Обучение создано, ожидает проведения |
| **Проводится** | Жёлтый | Обучение идёт прямо сейчас |
| **Завершено** | Зелёный | Обучение проведено |
| **Отменено** | Серый | Обучение отменено |

```
Запланировано → Проводится → Завершено
      ↓
  Отменено (из любого статуса)
```

### Как отметить обучение завершённым

1. Откройте детальную страницу обучения
2. Нажмите кнопку **«Завершить обучение»**
3. При необходимости можно приложить подпись (электронную)
4. Статус меняется на «Завершено», фиксируется дата и время

### Список обучений

**Что видно в таблице:**

| Колонка | Что показывает |
|---------|---------------|
| Дата | Когда проводится |
| Название | Тема обучения (с подтемами) |
| Тип | Цветной бейдж |
| Статус | Цветной бейдж |
| Инструктор | ФИО проводящего |
| Участники | Количество |
| Длительность | В минутах |

**Вкладки:** Все / Запланированные / Завершённые / Отменённые

**Показатели (вверху, 4 карточки):**
- Всего обучений
- Всего участников
- Всего часов обучения
- Просроченных (требуют внимания)

## Часть 2: Инструктажи

### 5 видов инструктажей (по ГОСТ 12.0.004-2015)

| Вид | Описание | Когда проводится | Периодичность |
|-----|---------|-----------------|---------------|
| **Вводный** | Общие правила безопасности организации. Проводится инженером по ОТ | При приёме на работу | 1 раз |
| **Первичный** | Правила на конкретном рабочем месте. Проводится непосредственным руководителем | Перед началом работ на новом месте | 1 раз на каждом рабочем месте |
| **Повторный** | Повторение первичного инструктажа. Те же темы | Каждые 6 месяцев (или чаще для опасных работ) | Регулярно |
| **Внеплановый** | После инцидента, при изменении правил, при новом оборудовании | По необходимости | Когда возникла причина |
| **Целевой** | Перед конкретной опасной работой (работа на высоте, в колодце, горячие работы) | Каждый раз перед работой | Каждый раз |

**Цвета типов:**
- Вводный — синий
- Первичный — зелёный
- Повторный — жёлтый
- Внеплановый — оранжевый
- Целевой — фиолетовый

> **Важно:** Без вводного и первичного инструктажа работник **не допускается** к работе. Это требование ТК РФ (ст. 225).

### Как провести инструктаж

1. Перейдите **Безопасность → Инструктажи**
2. Нажмите **«+ Новый инструктаж»**

3. Заполните:

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Тип инструктажа** | Да | Вводный / Первичный / Повторный / Внеплановый / Целевой |
| **Дата** | Да | Когда проводится |
| **Инструктор** | Да | Кто проводит (ФИО) |
| **Тема** | Да | О чём инструктаж |
| **Проект** | Нет | На каком объекте |
| **Примечание** | Нет | Дополнительная информация |
| **Участники** | Нет | Список сотрудников, которые проходят инструктаж |

4. Нажмите **«Создать»** → инструктаж в статусе «Запланирован»

### Как отметить инструктаж в системе

После проведения инструктажа **каждый участник должен расписаться**:

1. Откройте детальную страницу инструктажа
2. В разделе **«Участники»** видна таблица:

| Участник | Подпись | Дата подписи |
|----------|---------|-------------|
| Иванов И.И. | ✅ Подписано | 05.03.2026 14:30 |
| Петров П.П. | ❌ Не подписано | — |
| Сидоров С.С. | ✅ Подписано | 05.03.2026 14:35 |

3. Нажмите кнопку **«Подписать»** напротив каждого участника
4. Участник расписывается (электронная подпись)
5. Когда все расписались — нажмите **«Завершить инструктаж»**

> **Итого:** Внизу таблицы показано «3 из 5 подписали» — наглядный прогресс.

### Статусы инструктажа

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Запланирован** | Синий | Инструктаж создан, ожидает проведения |
| **Проводится** | Жёлтый | Инструктаж идёт, участники подписывают |
| **Завершён** | Зелёный | Все подписали, инструктаж завершён |
| **Отменён** | Серый | Инструктаж отменён |

### Следующий инструктаж

Для **повторного** инструктажа система показывает дату следующего планового инструктажа. Если дата близко (менее 30 дней) — отображается предупреждение. Если просрочено — красный текст «Просрочено».

### Список инструктажей

**Что видно в таблице:**

| Колонка | Что показывает |
|---------|---------------|
| Дата | Когда проводится |
| Тип | Цветной бейдж (5 видов) |
| Тема | О чём инструктаж |
| Инструктор | ФИО |
| Участники | Подписанные / Всего (например, «3/5») |
| Статус | Цветной бейдж |
| Следующий инструктаж | Дата следующего (для повторных) |

**Вкладки:** Все / Запланированные / Завершённые

**Показатели (вверху, 4 карточки):**
- Всего инструктажей
- Всего участников
- Завершено
- Просроченных (повторные, срок которых прошёл)

## Часть 3: Сертификаты и удостоверения работников

**Путь:** Безопасность → **Сертификаты работников**

Здесь хранятся все профессиональные удостоверения и допуски каждого сотрудника.

### Типы сертификатов

| Тип | Что это | Кто выдаёт | Срок действия |
|-----|---------|-----------|--------------|
| **ОТ и ТБ** | Охрана труда и техника безопасности | Учебные центры | 3 года |
| **Пожарная безопасность** | Пожарно-технический минимум | МЧС, учебные центры | 3 года |
| **Электробезопасность** | Группа допуска (2–5) | Ростехнадзор | 1 год |
| **Работа на высоте** | Допуск к работам на высоте | Учебные центры | 3 года |
| **Стропальщик** | Допуск к строповке грузов | Ростехнадзор | 1 год |
| **Допуск СРО** | Членство в саморегулируемой организации | СРО | Бессрочно (при членстве) |
| **НАКС (сварщик)** | Национальное агентство контроля сварки | НАКС | 2 года |
| **Промышленная безопасность** | Аттестация по промбезопасности | Ростехнадзор | 5 лет |

### Поля сертификата

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Сотрудник** | Да | Кому принадлежит |
| **Тип** | Да | Из списка выше |
| **Номер** | Нет | Номер удостоверения |
| **Дата выдачи** | Да | Когда получен |
| **Действует до** | Нет | Если не указано — бессрочный |
| **Кем выдан** | Нет | Организация |
| **Примечание** | Нет | Дополнительная информация |

### Статусы сертификата

| Статус | Цвет | Условие |
|--------|------|---------|
| **Действует** | Зелёный | До истечения > 90 дней |
| **Внимание** | Жёлтый | До истечения 30–90 дней |
| **Предупреждение** | Оранжевый | До истечения 15–30 дней |
| **Критично** | Красный | До истечения < 15 дней |
| **Истёк** | Тёмно-красный | Срок действия прошёл |

> Система автоматически проверяет сроки каждый день и предупреждает о истекающих сертификатах.

## Типичные сценарии

### Сценарий 1: Приём нового сотрудника

```
1. HR оформляет Иванова на работу (карточка сотрудника)

2. Инженер по ОТ проводит ВВОДНЫЙ инструктаж:
   → Тип: Вводный
   → Тема: Правила внутреннего распорядка, общие требования безопасности
   → Иванов подписывает → Статус: Завершён

3. Бригадир проводит ПЕРВИЧНЫЙ инструктаж на рабочем месте:
   → Тип: Первичный
   → Тема: Безопасность при кладочных работах
   → Иванов подписывает → Статус: Завершён

4. Иванов допущен к работе ✅

5. Через 6 месяцев — ПОВТОРНЫЙ инструктаж:
   → Система напоминает: «Повторный инструктаж для Иванова через 5 дней»
   → Бригадир проводит повторный инструктаж
```

### Сценарий 2: После инцидента

```
1. На площадке упал кирпич (инцидент зарегистрирован)

2. Инженер по ОТ проводит ВНЕПЛАНОВЫЙ инструктаж:
   → Тип: Внеплановый
   → Тема: Безопасное складирование материалов на перекрытиях
   → Все работники секции А подписывают

3. В карточке инструктажа указана ссылка на инцидент
```

### Сценарий 3: Перед опасной работой

```
1. Завтра — демонтаж перекрытия 5-го этажа

2. Прораб проводит ЦЕЛЕВОЙ инструктаж:
   → Тип: Целевой
   → Тема: Демонтаж ж/б конструкций, зона опасности, СИЗ
   → Все участники работ подписывают

3. Только после подписания — работы начинаются
```

## Частые вопросы

**Какие инструктажи обязательны?**
Вводный и первичный — обязательны для **каждого** нового работника. Повторный — каждые 6 месяцев. Без них работник не имеет права работать.

**Кто проводит вводный инструктаж?**
Инженер по охране труда или специалист, назначенный приказом руководителя.

**Кто проводит первичный инструктаж?**
Непосредственный руководитель (бригадир, мастер, прораб) на конкретном рабочем месте.

**Как часто нужен повторный инструктаж?**
Не реже одного раза в 6 месяцев. Для опасных работ — чаще (раз в 3 месяца).

**Когда проводится внеплановый инструктаж?**
После инцидента, при изменении технологии, при новом оборудовании, по требованию надзорных органов, при перерыве в работе более 60 дней.

**Обязательна ли подпись работника?**
Да. Без подписи инструктаж считается не пройденным. В системе подпись фиксируется электронно с датой и временем.

**Где хранятся записи об инструктажах?**
В журнале обучений (Безопасность → Журнал обучений). Каждая запись содержит: кто, когда, какой тип, кто проводил, когда следующий.

**Что будет, если пропустить повторный инструктаж?**
Система показывает предупреждение «Просрочено». Работник формально не допущен к работе. При проверке инспектором — штраф.

**Как проверить, все ли сертификаты в порядке?**
Откройте **Матрица сертификаций** — там видно по каждому сотруднику, какие допуски действуют, какие скоро истекут, а какие уже просрочены.
