# Полевая документация (наряд-допуски, лабораторные испытания, акты скрытых работ)

---

## Что это и зачем

**Полевая документация** — комплекс документов, которые оформляются непосредственно на строительной площадке. Без них невозможно сдать объект — инспекция проверяет каждый документ.

Три основных типа:
- **Наряд-допуски** — разрешение на опасные работы
- **Лабораторные испытания** — проверка качества материалов и конструкций
- **Акты скрытых работ** — фиксация работ, которые будут закрыты (засыпаны, залиты, заштукатурены)

## Часть 1: Наряд-допуски

### Что это

Наряд-допуск — письменное разрешение на выполнение работ повышенной опасности. Без наряда — работать НЕЛЬЗЯ. Штраф + уголовная ответственность при несчастном случае.

### Когда нужен наряд-допуск

| Вид работ | Нормативный акт |
|-----------|----------------|
| **Работы на высоте** (более 1,8 м) | Приказ Минтруда №782н |
| **Огневые работы** (сварка, резка) | ППР (правила противопожарного режима) |
| **Работы в замкнутых пространствах** | Приказ Минтруда №902н |
| **Земляные работы** (траншеи, котлованы) | СНиП 12-04-2002 |
| **Работы вблизи ЛЭП** | ПОТ Р М-016-2001 |
| **Работы с грузоподъёмными механизмами** | ФНП ПБ |
| **Демонтажные работы** | ПОС проекта |
| **Работы в зоне движения транспорта** | ПДД, ПОС |

### Содержание наряда-допуска

| Раздел | Что указывается |
|--------|----------------|
| **Кому** | ФИО ответственного производителя работ |
| **Что** | Какие работы выполняются |
| **Где** | Точное место (здание, ось, отметка) |
| **Когда** | Дата начала и окончания, время |
| **Условия** | Меры безопасности, СИЗ, ограждения |
| **Допуск** | Подписи: выдал / получил / допустил |
| **Ежедневный допуск** | Подпись каждый день перед началом работ |
| **Закрытие** | Работы завершены, место убрано, наряд закрыт |

> **Важно:** Наряд-допуск действует максимум 15 дней. После — оформляется новый.

## Часть 2: Лабораторные испытания

### Что это

Лабораторные испытания — проверка качества материалов и конструкций в аккредитованной лаборатории. Результат — протокол испытаний, который входит в исполнительную документацию.

### Что испытывают на стройке

| Материал / Конструкция | Что проверяют | Метод |
|-----------------------|---------------|-------|
| **Бетон** | Прочность (марка/класс) | Кубики 100×100 мм → пресс |
| **Арматура** | Прочность на разрыв, загиб | Растяжение, изгиб |
| **Грунт основания** | Несущая способность, состав | Зондирование, гранулометрия |
| **Сварные соединения** | Качество шва | УЗК, рентген, ВИК |
| **Раствор** | Прочность, подвижность | Кубики → пресс |
| **Кирпич / блоки** | Прочность, морозостойкость | Сжатие, заморозка-оттаивание |
| **Бетонные конструкции** | Прочность в конструкции | Склерометр (молоток Шмидта) |
| **Щебень / песок** | Фракция, примеси | Просеивание, промывка |

### Протокол испытаний

| Поле | Описание |
|------|----------|
| **Номер протокола** | Уникальный номер лаборатории |
| **Дата** | Когда проведено испытание |
| **Объект** | Название проекта |
| **Материал** | Что испытывали |
| **Место отбора** | Где взяли пробу (ось, отметка) |
| **Партия** | Номер партии / паспорт качества |
| **Результат** | Числовые значения (МПа, мм и т.д.) |
| **Соответствие** | Соответствует / Не соответствует ГОСТ/СНиП |
| **Лаборатория** | Наименование, аттестат аккредитации |

> **Совет:** Если бетон не набрал прочность — не нагружайте конструкцию! Ждите повторного испытания через 7/14/28 дней.

## Часть 3: Акты скрытых работ (АОСР)

### Что это

**АОСР** (Акт освидетельствования скрытых работ) — документ, который подтверждает, что скрытые работы выполнены качественно и могут быть закрыты (засыпаны, залиты, заштукатурены). Без АОСР — следующий этап работ начинать НЕЛЬЗЯ.

### Когда оформляется АОСР

| Скрытая работа | Что закрывает | Пример |
|----------------|--------------|--------|
| **Основание фундамента** | Котлован → бетон | Грунт проверен, можно заливать |
| **Армирование** | Арматура → бетон | Каркас собран правильно, можно заливать |
| **Гидроизоляция** | Изоляция → обратная засыпка | Мембрана уложена без повреждений |
| **Утепление** | Утеплитель → штукатурка | Толщина и крепление в норме |
| **Прокладка труб** | Трубы → стяжка/штукатурка | Трубы опрессованы, можно закрывать |
| **Электропроводка** | Кабели → штукатурка | Сопротивление замерено |

### Содержание АОСР

| Раздел | Что указывается |
|--------|----------------|
| **Объект** | Название, адрес |
| **Работы** | Какие скрытые работы выполнены |
| **Место** | Оси, отметки, помещения |
| **Проект** | Номер чертежа, по которому выполнялись работы |
| **Материалы** | Какие материалы использованы + сертификаты |
| **Результат осмотра** | Замечания или «соответствует проекту» |
| **Разрешение** | «Разрешается выполнение последующих работ» |
| **Подписи** | Подрядчик + заказчик (+ авторский надзор) |

> **Важно:** АОСР подписывают ДО того, как работы будут закрыты. Если залили бетон без АОСР на армирование — придётся вскрывать или проводить дорогостоящее обследование.

---

## Типичный сценарий

```
Ситуация: Монолитное перекрытие 3-го этажа

1. Установлена опалубка и арматура
2. Прораб вызывает представителя заказчика
3. Совместный осмотр: арматура соответствует чертежам ✓
4. Оформляется АОСР на армирование → подписи обеих сторон
5. Заливка бетона, отбор проб (кубики) → в лабораторию
6. Через 7 дней: промежуточные результаты (70% прочности)
7. Через 28 дней: протокол испытаний (100% прочности) ✓
8. Все документы загружены в систему → исполнительная документация
```

## Частые вопросы

**Можно ли оформить АОСР задним числом?**
Формально — нет. Практически — бывает, но это риск. Если заказчик откажется подписать (не видел реальное состояние) — будут проблемы при сдаче объекта.

**Кто оплачивает лабораторные испытания?**
Обычно подрядчик (входит в накладные расходы). Но заказчик может заказать дополнительные испытания за свой счёт.

**Сколько хранить наряд-допуски?**
30 дней после закрытия — для лёгких НС. 45 лет — если был тяжёлый/смертельный НС. Система хранит бессрочно.
