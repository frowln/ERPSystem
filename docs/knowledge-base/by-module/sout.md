# СОУТ — Специальная оценка условий труда

---

## Что это и зачем

**СОУТ** (специальная оценка условий труда) — обязательная процедура, при которой эксперт оценивает условия на каждом рабочем месте и присваивает класс вредности. От результатов СОУТ зависят доплаты работникам, дополнительные отпуска и взносы в пенсионный фонд.

Зачем в системе:
- **Учёт рабочих мест** — какие прошли СОУТ, какие нет
- **Контроль сроков** — СОУТ действует 5 лет, потом повторная
- **Класс условий** — влияет на льготы и компенсации сотрудникам
- **Штрафы** — без СОУТ штраф до 80 000 руб. за первое нарушение

## Классы условий труда

| Класс | Название | Описание | Компенсации |
|-------|----------|----------|-------------|
| **1** | Оптимальные | Вредные факторы отсутствуют | Нет |
| **2** | Допустимые | Факторы в пределах нормы | Нет |
| **3.1** | Вредные (1 степень) | Небольшое превышение нормы | Доплата 4%+ |
| **3.2** | Вредные (2 степень) | Начальные изменения здоровья | + доп. отпуск 7+ дней |
| **3.3** | Вредные (3 степень) | Стойкие изменения здоровья | + сокращённая неделя 36 ч |
| **3.4** | Вредные (4 степень) | Тяжёлые формы заболеваний | Все льготы |
| **4** | Опасные | Угроза жизни | Все льготы + доп. взносы |

### Типичные классы на стройке

| Профессия | Типичный класс | Вредные факторы |
|-----------|---------------|-----------------|
| Прораб (офис на площадке) | 2 | — |
| Каменщик | 3.1 | Тяжесть, пыль |
| Сварщик | 3.2 – 3.3 | Аэрозоли, излучение, шум |
| Маляр | 3.1 – 3.2 | Химические вещества |
| Крановщик | 3.1 | Вибрация, напряжённость |
| Работник в котловане | 3.2+ | Пыль, тяжесть, микроклимат |

## Процедура СОУТ

```
1. Создать комиссию по СОУТ (приказ руководителя)
2. Определить перечень рабочих мест для оценки
3. Заключить договор с аккредитованной лабораторией
4. Эксперт лаборатории проводит замеры на каждом рабочем месте:
   - Шум, вибрация, освещённость
   - Пыль, химические вещества
   - Тяжесть и напряжённость труда
   - Микроклимат (температура, влажность)
5. Лаборатория выдаёт отчёт с классами для каждого места
6. Комиссия подписывает отчёт
7. Результаты вносятся в систему
8. Работники ознакомляются под подпись
9. Данные передаются в ФСС (через ГИС СОУТ)
```

> **Важно:** СОУТ проводит только аккредитованная организация. Самостоятельно — нельзя.

## Частые вопросы

**Обязательна ли СОУТ?**
Да, для ВСЕХ работодателей. Исключение — дистанционные работники и религиозные организации. Штраф за отсутствие: 60-80 тыс. руб. для юрлица (первичное), 100-200 тыс. руб. (повторное).

**Как часто проводить?**
Каждые 5 лет. Внеплановая — при создании нового рабочего места, после несчастного случая, при изменении технологии.

**Можно ли снизить класс вредности?**
Да, если выдать работникам эффективные СИЗ (средства индивидуальной защиты). Класс может быть снижен на одну ступень.
