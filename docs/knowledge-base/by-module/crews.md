# Бригады

---

## Что это и зачем

Бригада — это группа рабочих под руководством бригадира, закреплённая за объектом. Например: «Бригада Иванова — 8 человек, кладка, объект "Ленина, 15"».

Работа на стройке организована бригадами. [Табель](timesheets.md), наряды, зарплата — всё привязано к бригаде. Без бригад невозможно:
- Закрепить группу людей за объектом
- Вести учёт рабочего времени бригадой
- Рассчитать стоимость работ по бригаде
- Формировать наряд-заказы

---

## Где найти

**Путь:** Главное меню → **Кадры** → **Бригады**

---

## Список бригад

**Что видно в таблице:**

| Колонка | Что показывает |
|---------|---------------|
| Название | Имя бригады (например, «Бригада каменщиков №1») |
| Бригадир | ФИО руководителя бригады |
| Кол-во рабочих | Сколько человек в бригаде |
| Специализация | Тип работ (кладка, электрика, сантехника и т.д.) |
| Текущий проект | На каком объекте сейчас работает |
| Статус | Активна / Свободна / В отпуске / Расформирована |
| Эффективность | Рейтинг работы (0–100%) |

**Показатели (вверху):**

| Показатель | Что значит |
|-----------|-----------:|
| Всего бригад | Общее количество |
| Всего рабочих | Сумма людей во всех бригадах |
| Активных | Бригады, работающие на объектах |
| Свободных | Бригады без назначения (можно направить на проект) |

---

## Статусы бригады

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Активна** | Зелёный | Работает на объекте |
| **Свободна** | Серый | Не назначена на проект, доступна |
| **В отпуске** | Синий | Вся бригада на отдыхе |
| **Расформирована** | Красный | Бригада больше не существует (только чтение) |

---

## Как создать бригаду

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Название** | Да | Например: «Бригада электриков №2» (до 255 символов) |
| **Бригадир** | Нет | ФИО руководителя бригады |
| **Телефон бригадира** | Нет | Контактный номер |
| **Кол-во рабочих** | Нет | Сколько человек в бригаде |
| **Специализация** | Нет | Тип работ: кладка, электрика, сантехника, бетон и т.д. |
| **Статус** | Да | По умолчанию: «Свободна» |

## Назначение на проект

Бригаду можно закрепить за проектом. Для этого:

1. Откройте страницу **Бригады**
2. Нажмите **«Назначить на проект»** на нужной бригаде
3. В модальном окне:
   - Выберите проект
   - Укажите дату начала
   - При необходимости — комментарий
4. Нажмите **«Назначить»**

Бригада получает статус **«Активна»** и отображается на странице проекта.

### Назначение отдельных сотрудников

Помимо бригад, можно назначать отдельных [сотрудников](employees.md) на проекты:

| Поле | Описание |
|------|---------|
| Сотрудник | Кого назначаем |
| Проект | На какой объект |
| Роль | Должность на этом проекте (может отличаться от основной) |
| Дата начала | С какого числа |
| Дата окончания | До какого числа (пусто = бессрочно) |
| Ставка | Ставка на этом проекте (может отличаться от основной) |

**Правила:**
- Один сотрудник не может быть назначен на один проект дважды
- Сотрудник может работать на нескольких проектах одновременно
- При снятии с проекта назначение помечается как неактивное (не удаляется)

## Частые вопросы

**Чем бригада отличается от назначения на проект?**
Бригада — это постоянная группа людей со своим бригадиром и специализацией. Назначение на проект — это привязка конкретного сотрудника к конкретному проекту с ролью и ставкой. Бригада может быть назначена на проект целиком.

**Можно ли одну бригаду назначить на несколько проектов?**
В системе у бригады одно поле «текущий проект». Для работы на нескольких объектах — используйте назначение отдельных сотрудников на проекты.

**Как расформировать бригаду?**
Измените статус на «Расформирована». Бригада останется в системе для истории, но не будет доступна для назначений.

**Где посмотреть всех людей на объекте?**
**Кадры → Бригады** → фильтр по проекту. Или в карточке проекта — вкладка «Команда».
