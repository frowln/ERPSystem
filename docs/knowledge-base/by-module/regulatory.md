# Разрешения, лицензии, СРО

---

## Что это и зачем

Строительство — одна из самых зарегулированных отраслей. Чтобы начать работы, нужно получить **разрешение на строительство**. Чтобы выполнять определённые виды работ — нужна **лицензия** или членство в **СРО**. А в процессе строительства приходят **проверки** от надзорных органов.

Без системного учёта:
- Разрешение на строительство истекло, а никто не заметил — штраф и остановка работ
- Инспектор Ростехнадзора пришёл с проверкой, а документы не готовы
- Предписание надо исполнить за 10 дней, а про него забыли
- Лицензия СРО заканчивается через месяц, а продление занимает 3 месяца

Модуль «Регуляторика» в системе ПРИВОД решает все эти задачи:
- **Разрешения** — реестр всех разрешительных документов с отслеживанием сроков
- **Лицензии и СРО** — учёт допусков к видам работ
- **Проверки** — календарь инспекций, подготовка, протоколы
- **Предписания** — контроль исполнения замечаний надзорных органов
- **Отчётность** — календарь сроков сдачи отчётов в госорганы

## Часть 1: Разрешения на строительство

**Разрешение** — документ, подтверждающий право выполнять определённые работы. Без разрешения работы **незаконны**.

### Типы разрешений

| Тип | Описание | Кто выдаёт |
|-----|----------|------------|
| **Разрешение на строительство** (BUILDING_PERMIT) | Основной документ для начала строительства | Администрация города/района |
| **Разрешение на снос** (DEMOLITION) | Право на демонтаж здания | Администрация |
| **Ордер на земляные работы** (EXCAVATION) | Право копать (вскрытие грунта, траншеи) | Администрация, ГАТИ |
| **Экологическое разрешение** (ENVIRONMENTAL) | Подтверждение экологической безопасности | Росприроднадзор |
| **Пожарная безопасность** (FIRE_SAFETY) | Соответствие нормам пожарной безопасности | МЧС |
| **Санитарное разрешение** (SANITARY) | Соответствие санитарным нормам | Роспотребнадзор |
| **Разрешение Ростехнадзора** (ROSTECHNADZOR) | Для опасных производственных объектов | Ростехнадзор |
| **Прочие** (OTHER) | Любые другие разрешительные документы | Разные органы |

### Статусы разрешения

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Черновик** (DRAFT) | Серый | Документ готовится |
| **Подано** (SUBMITTED) | Синий | Заявление подано в орган |
| **На рассмотрении** (UNDER_REVIEW) | Жёлтый | Орган рассматривает заявление |
| **Выдано** (APPROVED) | Зелёный | Разрешение получено ✅ |
| **Действует** (ACTIVE) | Зелёный | Разрешение в силе, работы можно вести |
| **Истекло** (EXPIRED) | Красный | Срок действия закончился — нужно продлевать |

```
Черновик → Подано → На рассмотрении → Выдано → Действует → Истекло
                                         ↓
                                      Отказано
```

### Что содержит карточка разрешения

| Поле | Описание |
|------|----------|
| **Тип** | Из списка выше (8 типов) |
| **Номер** | Регистрационный номер документа |
| **Дата выдачи** | Когда выдано |
| **Действует до** | Когда истекает |
| **Выдавший орган** | Кто выдал (название организации) |
| **Проект** | К какому проекту относится |
| **Ответственный** | Кто следит за сроками |
| **Скан документа** | Прикреплённый файл (PDF/фото) |

> **Важно:** Система предупреждает за 30, 14 и 7 дней до истечения разрешения. Красный индикатор на дашборде — разрешение просрочено.

## Часть 2: Лицензии

**Лицензия** — разрешение на определённый вид деятельности. Без лицензии выполнять такие работы **запрещено**.

### Типы лицензий

| Тип | Описание | Пример |
|-----|----------|--------|
| **СРО Строительство** (SRO_CONSTRUCTION) | Допуск к строительным работам | Членство в СРО «НОСТРОЙ» |
| **СРО Проектирование** (DESIGN) | Допуск к проектным работам | Членство в СРО «НОПРИЗ» |
| **СРО Инженерные изыскания** (ENGINEERING) | Допуск к изысканиям | Геодезия, геология, экология |
| **Специальное разрешение** (SPECIAL_PERMIT) | Работа с опасными объектами | Ростехнадзор, МЧС |

### Статусы лицензии

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Действует** (ACTIVE) | Зелёный | Лицензия актуальна |
| **Истекает** (EXPIRING) | Жёлтый | Скоро истечёт (менее 60 дней) |
| **Истекла** (EXPIRED) | Красный | Срок действия закончился |
| **Приостановлена** (SUSPENDED) | Оранжевый | Временно приостановлена |
| **Аннулирована** (REVOKED) | Красный | Отозвана навсегда |

## Часть 3: СРО (саморегулируемые организации)

**СРО** — это объединение строительных компаний, которое выдаёт допуски к определённым видам работ. С 2017 года членство в СРО обязательно для генподрядчиков (контракты свыше 10 млн ₽).

### Что хранится в системе

| Поле | Описание |
|------|----------|
| **Название СРО** | Например: «Ассоциация строителей Москвы» |
| **Регистрационный номер** | Номер в реестре СРО |
| **Допуск к видам работ** | Перечень работ, которые можно выполнять |
| **Компенсационный фонд** | Размер взноса в фонд СРО |
| **Дата вступления** | Когда вступили |
| **Действует до** | Когда нужно подтвердить членство |
| **Статус** | Действует / Приостановлен / Исключён |

> **Важно:** Если членство в СРО приостановлено — компания **не имеет права** заключать контракты на строительство свыше 10 млн ₽.

## Часть 4: Проверки (инспекции)

На стройку регулярно приходят проверки от различных надзорных органов. Модуль помогает **готовиться** к ним и **фиксировать** результаты.

### Типы проверок

| Тип | Описание | Когда |
|-----|----------|-------|
| **Плановая** (PLANNED) | По утверждённому графику | Раз в квартал/год, известно заранее |
| **Внеплановая** (UNPLANNED) | По жалобе или происшествию | Приходит неожиданно (но с уведомлением) |
| **Повторная** (FOLLOW_UP) | Проверка исполнения предписания | После устранения замечаний |

### Статусы проверки

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Запланирована** (SCHEDULED) | Синий | Дата назначена, готовимся |
| **В процессе** (IN_PROGRESS) | Жёлтый | Инспектор на объекте |
| **Завершена** (COMPLETED) | Зелёный | Проверка окончена, ждём результат |
| **Пройдена** (PASSED) | Зелёный | Замечаний нет ✅ |
| **Не пройдена** (FAILED) | Красный | Есть замечания, выдано предписание |
| **Отменена** (CANCELLED) | Серый | Проверка не состоялась |

### Какие органы проверяют стройку

| Орган | Что проверяет |
|-------|---------------|
| **ГИТ** (Государственная инспекция труда) | Охрана труда, условия работы |
| **Ростехнадзор** | Опасные объекты, грузоподъёмные механизмы |
| **Стройнадзор** | Соответствие проекту, качество работ |
| **МЧС** | Пожарная безопасность |
| **Роспотребнадзор** | Санитарные нормы, шум, пыль |
| **Росприроднадзор** | Экология, отходы, выбросы |
| **Администрация** | Благоустройство, ограждение, порядок |

### Подготовка к проверке

Для каждой проверки система предлагает **чек-лист подготовки**:

| Категория | Примеры документов |
|-----------|-------------------|
| **Разрешительные** | Разрешение на строительство, ордер на земляные работы |
| **Проектные** | Рабочая документация, заключение экспертизы |
| **Исполнительные** | АОСР, журнал работ, сертификаты материалов |
| **Охрана труда** | Журналы инструктажей, протоколы обучения |
| **Пожарная безопасность** | Планы эвакуации, акты проверки средств тушения |
| **Кадровые** | Приказы, допуски, удостоверения |

Каждый пункт чек-листа имеет статус: **готов** ✅ / **не готов** ❌ / **не требуется** ➖

## Часть 5: Предписания

**Предписание** — официальный документ от надзорного органа с требованием устранить нарушения в определённый срок. За неисполнение — штрафы и остановка работ.

### Статусы предписания

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Получено** (RECEIVED) | Красный | Предписание получено от надзорного органа |
| **В работе** (IN_PROGRESS) | Жёлтый | Идёт устранение замечаний |
| **Частично исполнено** (PARTIALLY_COMPLETED) | Оранжевый | Часть замечаний устранена |
| **Исполнено** (COMPLETED) | Зелёный | Все замечания устранены |
| **На проверке** (UNDER_VERIFICATION) | Синий | Отправлен ответ, ждём подтверждение от органа |
| **Подтверждено** (VERIFIED) | Зелёный | Орган подтвердил устранение ✅ |
| **Просрочено** (OVERDUE) | Красный | Срок исполнения истёк, замечания не устранены ❌ |
| **Обжаловано** (APPEALED) | Фиолетовый | Предписание оспорено в суде или вышестоящем органе |

```
Получено → В работе → Частично исполнено → Исполнено → На проверке → Подтверждено
                                                              ↓
                                                          Просрочено
```

### Карточка предписания

| Поле | Описание |
|------|----------|
| **Номер** | Номер предписания |
| **Дата выдачи** | Когда получено |
| **Срок исполнения** | До какой даты устранить |
| **Надзорный орган** | Кто выдал (ГИТ, Ростехнадзор и т.д.) |
| **Инспектор** | ФИО проверяющего |
| **Объект** | К какому проекту/участку |
| **Пункты предписания** | Список замечаний с описанием |
| **Ответственный** | Кто устраняет |
| **Документы** | Скан предписания, фото нарушений, акт устранения |

> **Критически важно:** Просроченное предписание может привести к штрафу до 1 000 000 ₽ или остановке строительства. Система выделяет просроченные предписания красным цветом и отправляет напоминания.

## Часть 6: Календарь отчётности

Строительная компания обязана сдавать различную отчётность в государственные органы в определённые сроки.

### Виды отчётности

| Отчёт | Куда | Периодичность |
|-------|------|---------------|
| Форма С-1 (объёмы строительства) | Росстат | Ежемесячно |
| Форма С-2 (ввод в действие) | Росстат | Ежеквартально |
| Отчёт по охране труда | ГИТ | Ежеквартально |
| Экологическая отчётность | Росприроднадзор | Ежегодно |
| Производственный контроль | Роспотребнадзор | Ежегодно |
| Отчёт в ИСУП | Минстрой (ИСУП) | По графику проекта |

### Статусы отчёта

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Предстоит** (UPCOMING) | Синий | Срок ещё не наступил |
| **К сдаче** (DUE) | Жёлтый | Срок сдачи приближается |
| **Сдан** (SUBMITTED) | Зелёный | Отчёт сдан вовремя ✅ |
| **Просрочен** (OVERDUE) | Красный | Срок прошёл, отчёт не сдан ❌ |
| **Не требуется** (NOT_REQUIRED) | Серый | Для данного проекта не нужен |

Система показывает **календарь** с цветовой индикацией: зелёные — сдано, красные — просрочено, жёлтые — скоро срок.

| Цвет | Уровень | Описание |

## Типичные сценарии

### Сценарий 1: Получение разрешения на строительство

```
1. Компания начинает новый проект — ЖК «Южный парк»
   → Нужно разрешение на строительство

2. Ответственный создаёт запись в системе:
   → Тип: Разрешение на строительство
   → Статус: Черновик
   → Готовит пакет документов

3. Документы готовы → подано заявление в администрацию:
   → Статус: Подано (12.01.2026)

4. Через 7 рабочих дней → ответ:
   → Статус: Выдано → Действует
   → Номер: 77-2026-0234
   → Действует до: 12.01.2029 (3 года)

5. Система ставит напоминания:
   → За 6 месяцев: «Разрешение истекает 12.01.2029»
   → За 3 месяца: «Срочно продлить»
   → За 1 месяц: «КРИТИЧНО — разрешение истекает!»
```

### Сценарий 2: Проверка Ростехнадзора

```
1. Получено уведомление: плановая проверка Ростехнадзора
   → Дата: 15.03.2026
   → Предмет: надзор за грузоподъёмными механизмами

2. Ответственный создаёт проверку в системе:
   → Тип: Плановая
   → Орган: Ростехнадзор
   → Статус: Запланирована

3. Открывает чек-лист подготовки:
   → Паспорта кранов: готовы ✅
   → Журнал осмотра стропов: не заполнен ❌
   → Удостоверения стропальщиков: готовы ✅
   → Акт испытания тары: просрочен ❌

4. За 7 дней до проверки: устраняет пробелы
   → Все пункты: готовы ✅

5. День проверки:
   → Статус: В процессе
   → Инспектор: Петров А.И.
   → Результат: Пройдена ✅ (одно незначительное замечание)
```

### Сценарий 3: Исполнение предписания ГИТ

```
1. После проверки ГИТ выдано предписание:
   → Замечание 1: Нет ограждения проёма на 4 этаже
   → Замечание 2: У 3 рабочих просрочены удостоверения
   → Замечание 3: Не проведён повторный инструктаж
   → Срок исполнения: 10 рабочих дней (до 28.03.2026)

2. Ответственный за ОТ берёт в работу:
   → Статус: В работе
   → Замечание 1: Установлено ограждение (20.03) ✅
   → Замечание 2: Направлены на обучение (22.03) ✅
   → Замечание 3: Проведён инструктаж (23.03) ✅

3. Статус: Исполнено → отправлен ответ в ГИТ
   → Приложены фото устранения, копии удостоверений

4. ГИТ назначает повторную проверку:
   → Тип: Повторная (FOLLOW_UP)
   → Результат: Подтверждено ✅

5. Предписание: Подтверждено → закрыто
```

## Частые вопросы

**Что будет, если работать без разрешения на строительство?**
Штраф по КоАП: от 500 000 до 1 000 000 ₽ для юридических лиц. Плюс возможна принудительная остановка работ и снос.

**На какой срок выдаётся разрешение на строительство?**
На весь срок строительства по проекту организации строительства (ПОС). Обычно 2–5 лет. Можно продлить.

**Обязательно ли членство в СРО?**
Для генподрядчиков с контрактами свыше 10 млн ₽ — да. Для субподрядчиков — зависит от суммы и вида работ.

**Как часто приходят проверки?**
Плановые — по графику (обычно раз в год). Внеплановые — после жалоб, аварий, несчастных случаев. На крупных объектах Стройнадзор может приходить ежемесячно.

**Что делать с предписанием?**
Устранить все замечания в указанный срок и отправить письменный ответ с доказательствами. Если не успеваете — подать ходатайство о продлении срока (до его истечения).

**Можно ли оспорить предписание?**
Да, в течение 10 дней. Через вышестоящий орган или суд. Но до решения по жалобе предписание всё равно нужно исполнять.

**Что такое индекс соответствия?**
Это процентный показатель, который учитывает: действующие разрешения, наличие лицензий, исполненные предписания, сданную отчётность. 100% — всё в порядке, ниже 60% — серьёзные проблемы.
