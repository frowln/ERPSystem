# Мобильное приложение (PWA)

---

## Что это и зачем

**Мобильное приложение** — версия системы ПРИВОД, которая работает прямо в браузере телефона, но выглядит и ведёт себя как обычное приложение. Можно установить на главный экран, получать уведомления, работать даже без интернета.

Зачем это нужно:
- **Прораб на объекте** — отмечает выполнение работ, фотографирует дефекты, заполняет журнал
- **Инженер ОТ** — проводит обход, фиксирует нарушения, выдаёт предписания
- **Кладовщик** — принимает материалы, сканирует накладные
- **Руководитель** — смотрит дашборд и согласовывает документы в дороге

## Как установить

### На iPhone (Safari)

1. Откройте сайт системы в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой внизу)
3. Выберите «На экран "Домой"»
4. Нажмите «Добавить»

### На Android (Chrome)

1. Откройте сайт системы в Chrome
2. Нажмите три точки (меню) в правом верхнем углу
3. Выберите «Установить приложение» или «Добавить на главный экран»
4. Подтвердите

> **Важно:** После установки приложение открывается в полноэкранном режиме — без адресной строки браузера. Выглядит как обычное приложение.

## Что работает в мобильной версии

| Функция | Описание |
|---------|----------|
| **Все страницы** | Весь функционал доступен — те же модули, что на компьютере |
| **Адаптивный дизайн** | Интерфейс перестраивается под размер экрана |
| **Push-уведомления** | Уведомления о задачах, согласованиях, сообщениях |
| **Камера** | Фото дефектов, документов, объекта — прямо из системы |
| **Офлайн-режим** | Базовые функции работают без интернета |

---

## Офлайн-режим

Если на объекте нет интернета:

1. Данные, загруженные ранее, доступны для просмотра
2. Новые записи (дефекты, фото, записи журнала) сохраняются в телефоне
3. Когда интернет появится — данные автоматически отправятся на сервер
4. Если есть конфликты (кто-то изменил те же данные) — система предложит выбрать версию

> **Совет:** Перед выездом на объект без связи — откройте нужные страницы, чтобы данные загрузились в кэш.

## Push-уведомления

Какие уведомления приходят:

| Событие | Пример |
|---------|--------|
| **Новая задача** | «Вам назначена задача: Проверить армирование плиты» |
| **Согласование** | «Документ ожидает вашего согласования: КС-2 №14» |
| **Сообщение** | «Новое сообщение в проекте ЖК Южный» |
| **Срок** | «Задача просрочена: Устранить дефект №45» |
| **Предупреждение** | «Бюджет проекта превышен на 5%» |

Настройка уведомлений: **Настройки → Уведомления** — можно включить/выключить каждый тип.

## Рекомендации для работы с телефона

| Совет | Почему |
|-------|--------|
| Используйте Wi-Fi для загрузки больших файлов | Экономия мобильного трафика |
| Установите приложение на главный экран | Быстрый доступ, полноэкранный режим |
| Включите push-уведомления | Не пропустите важные согласования |
| Фотографируйте горизонтально | Лучше видно детали на фото дефектов |
| Синхронизируйте перед выездом | Свежие данные будут доступны офлайн |

## Отличие от компьютерной версии

Мобильная версия — это **та же самая система**, не упрощённая копия. Разница только в расположении элементов на экране:
- Таблицы прокручиваются горизонтально
- Боковое меню скрывается и открывается по кнопке
- Кнопки и поля крупнее для удобства нажатия пальцем

## Частые вопросы

**Нужно ли скачивать из App Store / Google Play?**
Нет. Это PWA — прогрессивное веб-приложение. Устанавливается из браузера. Не нужен аккаунт в магазине приложений.

**Работает ли на старых телефонах?**
Да, если телефон поддерживает современный браузер (Safari 14+, Chrome 80+). Это практически любой смартфон не старше 5 лет.

**Сколько места занимает?**
Около 10-20 МБ — в десятки раз меньше, чем обычное приложение.
