# Инспекции качества

---

## Что это и зачем

Инспекция качества — это проверка, что строительные работы и материалы соответствуют проекту, стандартам и нормативам. Без проверок качества объект не примут в эксплуатацию, а заказчик может отказаться платить за некачественные работы.

В системе ПРИВОД модуль «Качество» объединяет:
- **Проверки качества** — проверка конкретных работ или материалов
- **Чек-листы** — списки вопросов для систематической проверки
- **Сертификаты материалов** — паспорта и сертификаты на стройматериалы
- **Входной контроль** — приёмка материалов на складе
- **Допуски и отклонения** — правила: какие отклонения допустимы
- **Ворота качества** — контрольные точки: нельзя продолжать работу, пока не пройдена проверка

## Часть 1: Проверки качества

Проверка качества — основной инструмент контроля. Каждая проверка фиксирует: что проверяли, кто проверял, какой результат.

### Типы проверок

| Тип | Когда применяется | Пример |
|-----|------------------|--------|
| **Входной контроль материалов** | При поступлении материалов на стройку | Проверка арматуры: марка, диаметр, сертификат |
| **Промежуточная проверка работ** | Во время выполнения работ | Проверка армирования перед бетонированием |
| **Освидетельствование скрытых работ** | Перед закрытием скрытых конструкций | Гидроизоляция фундамента перед обратной засыпкой |
| **Итоговая проверка** | После завершения работ | Приёмка готового этажа |
| **Лабораторная** | Испытание образцов | Прочность бетонных кубиков на 28-й день |

### Как создать проверку

1. Перейдите **Качество → Список проверок**
2. Нажмите **«+ Новая проверка»**
3. Заполните:

| Поле | Обязательно? | Описание |
|------|-------------|----------|
| **Тип проверки** | Да | Один из 5 типов выше |
| **Название** | Да | Краткое описание (например: «Проверка арматуры плиты 3 этажа») |
| **Проект** | Да | К какому проекту относится |
| **Описание** | Нет | Подробности: что именно проверять |
| **Плановая дата** | Нет | Когда запланирована проверка |
| **Инспектор** | Нет | Кто проводит проверку |

4. Нажмите **«Создать»** — проверка в статусе «Запланирована»
5. Система автоматически присваивает номер: **QC-XXXXX** (например, QC-00042)

### Статусы проверки

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Запланирована** (PLANNED) | Синий | Проверка создана, ещё не начата |
| **В процессе** (IN_PROGRESS) | Жёлтый | Инспектор проводит проверку |
| **Завершена** (COMPLETED) | Зелёный | Проверка проведена, результат зафиксирован |

```
Запланирована → В процессе → Завершена
```

### Результаты проверки

При завершении проверки инспектор выставляет результат:

| Результат | Цвет | Что означает |
|-----------|------|-------------|
| **Принято** (PASS) | Зелёный | Работа/материал соответствует требованиям |
| **Принято с замечаниями** (CONDITIONAL_PASS) | Жёлтый | В целом ОК, но есть мелкие замечания |
| **Не принято** (FAIL) | Красный | Не соответствует. Нужно переделать |
| **Ожидание** (PENDING) | Серый | Результат пока не определён (ждём лабораторию и т.п.) |

## Часть 2: Чек-листы

Чек-лист — это стандартизированный список вопросов для проверки. Вместо того чтобы каждый раз вспоминать, что проверять — инспектор идёт по списку и отмечает каждый пункт.

### Типы работ для чек-листов

| Тип работ | Что проверяется |
|-----------|----------------|
| **Бетонирование** | Марка бетона, уплотнение, уход за бетоном |
| **Монтаж металлоконструкций** | Узлы крепления, качество сварки, антикоррозийка |
| **Сварочные работы** | Качество шва, катет, отсутствие трещин |
| **Гидроизоляция** | Непрерывность, нахлёст, адгезия |
| **Кровельные работы** | Уклоны, примыкания, водоотведение |
| **Отделочные работы** | Ровность стен, углы, качество покраски |
| **Электромонтаж** | Сопротивление изоляции, заземление, маркировка |
| **Сантехника** | Давление в трубах, уклоны, герметичность |
| **Теплоизоляция** | Толщина, непрерывность, крепление |
| **Кладка** | Перевязка, толщина шва, вертикальность |
| **Земляные работы** | Глубина, откосы, дно котлована |
| **Благоустройство** | Уклоны, дренаж, покрытие |
| **Прочее** | Любые другие работы |

### Как работать с чек-листом

1. Перейдите **Качество → Чек-листы**
2. Нажмите **«+ Новый чек-лист»** или выберите из шаблона
3. Заполните общую информацию:

| Поле | Описание |
|------|----------|
| **Тип работ** | Из списка выше (13 типов) |
| **Название** | Например: «Бетонирование плиты 2 этажа» |
| **Проект** | К какому проекту |
| **Описание** | Подробности |

4. Пройдите по каждому пункту чек-листа и отметьте:

| Оценка | Значение |
|--------|---------|
| **Соответствует** (PASS) | Пункт выполнен |
| **Не соответствует** (FAIL) | Пункт не выполнен — нарушение |
| **Не применимо** (N/A) | Этот пункт не относится к данной работе |

5. Добавьте комментарии к отдельным пунктам, если нужно
6. Завершите чек-лист — система рассчитает общий процент соответствия

### Шаблоны чек-листов

Чтобы каждый раз не создавать чек-лист с нуля, используйте шаблоны.

**Путь:** Качество → **Шаблоны чек-листов**

Шаблон содержит готовый набор пунктов для проверки конкретного типа работ. Например, шаблон «Бетонирование» может включать:
- Опалубка установлена по чертежу
- Арматура соответствует проекту
- Защитный слой обеспечен
- Закладные детали установлены
- Подготовка основания выполнена
- Марка бетона соответствует
- Бетон уложен без расслоения

Администратор или инженер по качеству создаёт шаблоны один раз, а потом все инспекторы используют их на всех проектах.

## Часть 3: Сертификаты материалов

Каждый строительный материал должен иметь сертификат (паспорт качества). Без сертификата материал нельзя использовать на объекте.

### Типы сертификатов

| Тип | Что подтверждает |
|-----|-----------------|
| **Сертификат качества** | Соответствие техническим условиям производителя |
| **Сертификат соответствия** | Соответствие ГОСТу или техрегламенту |
| **Сертификат происхождения** | Страна производства |
| **Протокол испытаний** | Результаты лабораторных испытаний |
| **Сертификат безопасности** | Пожарная, экологическая, санитарная безопасность |
| **Сертификат ГОСТ** | Соответствие конкретному ГОСТу |
| **Сертификат ISO** | Соответствие международному стандарту |

### Статусы сертификата

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Действителен** (VALID) | Зелёный | Сертификат актуален, можно использовать |
| **Истёк** (EXPIRED) | Серый | Срок действия закончился |
| **Отозван** (REVOKED) | Красный | Сертификат отозван (брак, нарушения) |
| **На проверке** (PENDING_VERIFICATION) | Жёлтый | Ожидает подтверждения подлинности |

### Что заполняется в сертификате

| Поле | Описание |
|------|----------|
| **Номер сертификата** | Уникальный номер документа |
| **Тип** | Из списка выше (7 типов) |
| **Выдан** | Кто выдал (лаборатория, производитель, орган сертификации) |
| **Дата выдачи** | Когда выдан |
| **Срок действия** | До когда действует |
| **Материал** | Какой материал сертифицирован |
| **Поставщик** | От кого получен |
| **ГОСТ / Стандарт** | На соответствие какому стандарту |

К сертификату можно прикрепить скан документа (PDF, фото).

## Часть 4: Входной контроль материалов

Когда материал приходит на стройку — его нужно проверить перед использованием. Это и есть входной контроль.

**Путь:** Качество → **Входной контроль**

### Как провести входной контроль

1. Нажмите **«+ Новый осмотр»**
2. Заполните:

| Поле | Описание |
|------|----------|
| **Материал** | Что проверяем |
| **Поставщик** | От кого получили |
| **Количество** | Сколько единиц |
| **Дата осмотра** | Когда проверяли |
| **Инспектор** | Кто проверял |

3. Проведите осмотр и зафиксируйте результат:

| Результат | Что означает |
|-----------|-------------|
| **Принято** | Материал соответствует, можно использовать |
| **Отклонено** | Не соответствует — вернуть поставщику |
| **Условно принято** | Есть отклонения, но можно использовать с ограничениями |

4. Запишите результаты осмотра (визуальный осмотр, замеры, сравнение с сертификатом)
5. При необходимости отправьте образцы в лабораторию

## Часть 5: Допуски и отклонения

Ни одна стройка не бывает идеальной — всегда есть отклонения от проекта. Вопрос: какие отклонения допустимы, а какие — нет.

**Путь:** Качество → **Допуски**

### Категории допусков

| Категория | Что контролируется | Пример |
|-----------|-------------------|--------|
| **Размерные** | Геометрические размеры | Отклонение стены от вертикали ≤ 10 мм на 3 м |
| **Конструктивные** | Несущие элементы | Отклонение оси колонны ≤ 5 мм |
| **Отделочные** | Качество отделки | Неровность пола ≤ 2 мм на 2 м |
| **Монтажные** | Монтаж оборудования | Отклонение от оси ≤ 3 мм |
| **Электрические** | Электроустановки | Сопротивление изоляции ≥ 0.5 МОм |
| **Сантехнические** | Трубопроводы | Давление при испытании ≥ 1.5 рабочего |

Каждое правило содержит:
- **Минимальное значение** — ниже нельзя
- **Номинальное значение** — то, что в проекте
- **Максимальное значение** — выше нельзя
- **Единица измерения** — мм, мм/м, МОм, бар и т.д.

При проверке система сравнивает фактическое значение с допуском и выдаёт результат: **Соответствует** / **Не соответствует** / **Требует перепроверки**.

## Часть 6: Ворота качества

**Ворота качества** (Quality Gate) — это контрольная точка в графике строительства. Пока ворота не «открыты» — следующий этап работ начинать нельзя.

**Путь:** Качество → **Ворота качества**

### Как работают ворота

Каждые ворота оценивают три параметра:

| Параметр | Что проверяется | Как считается |
|----------|----------------|---------------|
| **Документы** | Вся ли документация оформлена | % готовых документов от требуемых |
| **Проверки качества** | Все ли проверки пройдены | % пройденных проверок от запланированных |
| **Объёмы** | Выполнены ли физические объёмы | % выполнения от плана |

### Статусы ворот

| Статус | Цвет | Что означает |
|--------|------|-------------|
| **Не начато** | Серый | Ворота ещё не активированы |
| **В процессе** | Жёлтый | Идёт проверка условий |
| **Заблокировано** | Красный | Условия не выполнены — продолжать нельзя |
| **Пройдено** | Зелёный | Все условия выполнены — можно продолжать |
| **Не пройдено** | Красный | Проверка завершена, условия не выполнены |

### Пример ворот качества

```
Ворота: «Бетонирование плиты 3 этажа»

Условия:
 ✅ Документы: 100% (4 из 4)
    — Чертёж опалубки утверждён
    — Схема армирования согласована
    — Журнал бетонных работ открыт
    — Акт на скрытые работы (армирование) подписан

 ✅ Проверки: 100% (3 из 3)
    — Осмотр опалубки: Принято
    — Проверка армирования: Принято
    — Входной контроль бетона: Принято

 ✅ Объёмы: 100%
    — Армирование выполнено на 100%
    — Опалубка установлена на 100%

Результат: ПРОЙДЕНО ✅ → Можно бетонировать
```

## Типичные сценарии

### Сценарий 1: Входной контроль арматуры

```
1. На стройку приехала фура с арматурой ø12 мм, 20 тонн

2. Кладовщик проверяет документы:
   → Сертификат качества: есть ✅
   → Марка стали: А500С (как в проекте) ✅
   → ГОСТ: 34028-2016 ✅

3. Инженер по качеству проводит входной контроль:
   → Визуальный осмотр: коррозии нет, деформаций нет ✅
   → Замер диаметра: 12.1 мм (допуск ±0.5) ✅
   → Количество: 20 тонн ✅

4. Результат: «Принято»
   → Арматура отправляется на склад
   → Сертификат прикреплён в системе
   → Можно использовать в работе
```

### Сценарий 2: Проверка бетонирования по чек-листу

```
1. Бетонщики завершили заливку плиты 2 этажа

2. Инженер по качеству открывает чек-лист «Бетонирование»:

   ✅ Опалубка соответствует чертежу
   ✅ Арматура уложена по проекту
   ✅ Защитный слой 25 мм обеспечен
   ✅ Закладные детали установлены
   ✅ Марка бетона В25 (по накладной и паспорту)
   ❌ Бетон не провибрирован в зоне колонны К-4
   ✅ Поверхность ровная
   ✅ Уход за бетоном организован (укрытие плёнкой)

3. Результат: 7 из 8 = 87.5%
   → Результат проверки: «Принято с замечаниями»
   → Создаётся дефект: «Непровибрированный участок у К-4»
   → Назначается подрядчик для устранения
```

### Сценарий 3: Ворота качества перед штукатуркой

```
1. Стены 3 этажа выложены, нужно начинать штукатурку

2. Открываются ворота качества «Штукатурные работы — 3 этаж»:

   ❌ Документы: 75% (3 из 4)
      — Нет акта на скрытые работы (электропроводка в стенах)
   ✅ Проверки: 100%
      — Геодезическая съёмка стен: OK
      — Проверка кладки: OK
   ✅ Объёмы: 100%

3. Статус ворот: ЗАБЛОКИРОВАНО ❌
   → Штукатурку начинать нельзя

4. Подписывают акт на скрытые работы
   → Документы: 100%
   → Статус ворот: ПРОЙДЕНО ✅
   → Можно приступать к штукатурке
```

## Частые вопросы

**Кто проводит проверки качества?**
Инженер по качеству (QA/QC), инженер-строитель, авторский надзор (проектировщик), технический надзор (заказчик). У каждого своя зона ответственности.

**Обязательны ли чек-листы?**
По СП 48.13330 строительный контроль обязателен. Чек-листы — удобный способ его оформить и не забыть ни один пункт.

**Что делать, если проверка не пройдена?**
Создать дефект или NCR, назначить ответственного за устранение, после устранения — провести повторную проверку.

**Можно ли использовать материал без сертификата?**
Нельзя. По ГОСТ и СП использование материалов без документов о качестве запрещено. Это может привести к проблемам при сдаче объекта.

**Зачем нужны ворота качества?**
Чтобы не начинать следующий этап, пока предыдущий не завершён и не проверен. Это предотвращает переделки: штукатурить стену, в которой нет электропроводки — дорогая ошибка.

**Как часто нужно проводить проверки?**
Зависит от типа работ. Скрытые работы — перед закрытием (каждый раз). Бетонирование — при каждой заливке. Общий осмотр — минимум раз в неделю.

**Где хранятся фото проверок?**
Фотографии прикрепляются к проверке или чек-листу. Хранятся в системе и доступны при любой проверке или аудите.
