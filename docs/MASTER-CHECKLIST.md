## 4. Правильность расчётов

### Финансы
- [ ] НДС выделение: × 20/120
- [ ] НДС начисление: × 1.2
- [ ] Округление до копеек (2 знака)
- [ ] Маржа = доход - себестоимость
- [ ] Процент маржи = маржа / доход × 100

### Сметы
- [ ] Прямые затраты = (ОЗП + ЭМ + МАТ) × объём
- [ ] Накладные расходы по МДС 81-33.2004
- [ ] Сметная прибыль по МДС 81-25.2001
- [ ] НДС в конце на итог, не на каждую строку
- [ ] Коэффициенты Минстроя актуальные и обновляемые

### HR
- [ ] Табель: производственный календарь (не хардкод 168)
- [ ] Овертайм: первые 2ч × 1.5, далее × 2.0
- [ ] Ночные: +20%
- [ ] Выходные/праздничные: × 2.0
- [ ] Отпуск: 28 дней, расчёт через 29.3
- [ ] НДФЛ 13%
- [ ] Страховые: ОПС 22%, ОМС 5.1%, ОСС 2.9%

### Склад
- [ ] Баланс = приход - расход
- [ ] Не уходит в минус
- [ ] Метод списания реализован корректно
- [ ] ГСМ: норма vs факт

### EVM
- [ ] CPI = EV / AC
- [ ] SPI = EV / PV
- [ ] EAC, ETC, VAC

### Охрана труда
- [ ] LTIFR = (НС × 1 000 000) / отработанные часы

## 5. Безопасность

### Аутентификация
- [ ] Пароли: bcrypt/argon2 (не md5)
- [ ] JWT: короткий срок access, refresh token, отзыв
- [ ] Блокировка после 5 неверных попыток
- [ ] Сброс пароля — безопасный ли процесс

### Авторизация
- [ ] Матрица ролей × 2493 эндпоинта — все проверены
- [ ] @PreAuthorize на каждом эндпоинте
- [ ] ProtectedRoute на каждом роуте фронтенда
- [ ] Горизонтальная эскалация невозможна (подстановка чужого ID)
- [ ] Вертикальная эскалация невозможна (viewer → admin API)
- [ ] Портал: клиент не видит маржу, себестоимость, КЛ

### Инъекции
- [ ] SQL injection: параметризованные запросы везде
- [ ] XSS: нет dangerouslySetInnerHTML без санитайзинга
- [ ] CSRF: токены
- [ ] Path traversal: загрузка файлов безопасна

### Данные
- [ ] HTTPS принудительный
- [ ] FieldEncryptionService на: паспорт, СНИЛС, ИНН
- [ ] CORS: не стоит *
- [ ] Секреты не в git, через ${ENV_VAR}
- [ ] Зависимости без HIGH/CRITICAL уязвимостей

### Файлы
- [ ] Проверка типа, размера, содержимого при загрузке
- [ ] Исполняемые файлы нельзя загрузить
- [ ] Файлы не в публичной папке
