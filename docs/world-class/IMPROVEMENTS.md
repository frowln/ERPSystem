## Must Have (без этого не продашь)

1. [MUST] [INTEGRATION] Реальная 1С интеграция — двусторонняя синхронизация контрагентов, номенклатуры, документов, оплат. Без 1С российский строительный бизнес не купит ERP
2. [MUST] [INTEGRATION] ЭДО (Diadoc/SBIS) — электронный обмен документами с КЭП. Обязательно для B2B
3. [MUST] [PLANNING] EVM (Earned Value Management) — CPI, SPI, EAC, ETC. Строительные компании используют EVM для контроля проектов уровня 100M+ руб
4. [MUST] [PLANNING] Critical Path Method (CPM) — автоматическое определение критического пути в графике. Базовая функция любого строительного planning tool
5. [MUST] [WORKFLOW] Визуальный drag-and-drop редактор маршрутов согласования — условная маршрутизация, делегирование, эскалация. Согласования = ядро строительного бизнес-процесса
6. [MUST] [FORMS] Печатные формы М-29, С-3, АОСР, UPD, Т-12/Т-13 — юридическая обязанность в строительстве
7. [MUST] [SECURITY] SSO (SAML 2.0 / OIDC) — для enterprise клиентов это обязательное требование при закупке
8. [MUST] [PORTAL] Реальный портал подрядчика с отдельной авторизацией, получением нарядов, подачей КС-2
