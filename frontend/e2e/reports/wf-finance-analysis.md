## 1. Цифры сходятся на всех уровнях?

### Цепочка: счёт → платёж → КС-2 → КС-3 → бюджет

| Уровень | Статус | Детали |
|---------|--------|--------|
| Счёт → строки | ✅ PASS | SUM(строк) = итого на каждом из 3 счетов |
| НДС на каждом счёте | ✅ PASS | 20.00% ровно (23 400 / 44 400 / 15 200 / 107 980) |
| Платёж = счёт | ✅ PASS | 140 400 + 266 400 + 91 200 = 498 000 ₽ |
| КС-2 строки → итого | ✅ PASS | 182 000 + 217 500 + 140 400 = 539 900 ₽ |
| КС-3 = SUM(КС-2) | ✅ PASS | 539 900 ₽ |
| Исходящий счёт = КС-3 + НДС | ✅ PASS | 539 900 + 107 980 = 647 880 ₽ |
| Баланс = доход - расход | ✅ PASS | 647 880 - 498 000 = +149 880 ₽ |

**Вердикт**: Математика на API-уровне корректна. Все предварительно рассчитанные суммы совпадают. НДС = ровно 20% на каждом документе.

## 7. Time to close (оценка)

| Операция | Время в PRIVOD | Время в 1С | Комментарий |
|----------|---------------|------------|-------------|
| Создать 3 счёта | ~5 мин (API) / ~15 мин (UI) | ~10 мин | Наш UI формы конкурентоспособен |
| Оплатить 3 счёта | ~3 мин (API) / ~10 мин (UI) | ~5 мин | 1С быстрее (банк-клиент) |
| КС-2 + 3 строки | ~2 мин (API) / ~8 мин (UI) | ~15 мин | PRIVOD быстрее |
| КС-3 + связь | ~1 мин (API) / ~5 мин (UI) | ~10 мин | PRIVOD быстрее |
| Исходящий счёт | ~1 мин (API) / ~5 мин (UI) | ~5 мин | Паритет |
| Сверка | ~30 сек (API assert) | ~5 мин | Наш API даёт автопроверку |
| **ИТОГО** | **~12 мин (API) / ~45 мин (UI)** | **~50 мин** | Конкурентоспособно |

**Вердикт**: Полная финансовая цепочка (создание документов → оплата → закрытие → сверка) выполняется за ~45 минут через UI, что сопоставимо с 1С. API-путь значительно быстрее.
