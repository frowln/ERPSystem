### 1.2 Продукт: полный список модулей и функций

#### Управление планами (Plan Management)
- Просмотр 2D и 3D планов на любом устройстве
- OCR — автоматическое распознавание названий листов и деталей
- Сравнение планов (overlay tool) — наложение двух версий с выделением различий
- Версионирование листов (sheet versioning)
- Маркировка и аннотации в реальном времени (markups)
- Гиперссылки между листами (hyperlinks/detail callouts)
- Загрузка из облачных хранилищ (Box, Dropbox, OneDrive, Google Drive, SharePoint, Egnyte)

#### Управление задачами (Task Management)
- Создание и назначение задач с привязкой к чертежам
- Категории, хэштеги, приоритеты, дедлайны
- Чек-листы внутри задач (TaskCheckItems)
- Комментарии, фото, файлы в задачах (Bubbles)
- Отслеживание статусов и прогресса
- Фильтрация по категории, исполнителю, приоритету, статусу

#### Punch List (Снятие замечаний)
- Привязка дефектов к конкретному месту на чертеже
- Фотодокументирование
- Автоматические отчеты по punch-листам
- Категоризация дефектов

#### Формы (Forms)
- 6 отраслевых шаблонов: Daily Report, RFI, T&M, Inspection Request и др.
- Создание пользовательских шаблонов форм
- Цифровые подписи
- Секции и поля с различными типами ввода
- Экспорт форм в PDF

#### Планирование (Scheduling)
- Kanban-доска
- Календарное представление
- Диаграмма Ганта
- Импорт задач из Excel/CSV (тысячи задач за раз)
- Автоматическое отображение задач с датами на календаре

#### Отчётность (Reporting)
- Task Reports — отчёты по задачам
- Form Reports — отчёты по формам
- Автоматическая отправка отчётов (ежедневно/еженедельно/ежемесячно)
- Экспорт в PDF

#### RFI, Change Orders, Submittals (Business Plus)
- Создание и управление RFI
- Управление заявками на изменение
- Управление сабмитталами
- Совместная работа с субподрядчиками над документацией

#### Коммуникация
- Мгновенные сообщения в контексте проекта
- Обмен файлами
- Обновления в реальном времени
- Интеграция с Microsoft Teams — уведомления о статусах форм

#### Управление файлами
- Облачное хранилище документов
- Синхронизация с внешними хранилищами
- Контроль версий документов

### 1.3 Уникальные особенности

1. **OCR-распознавание чертежей** — автоматическое распознавание и именование листов, деталей и перекрёстных ссылок. Конкуренты этого, как правило, не предлагают.
2. **Сравнение планов (Plan Compare/Overlay)** — наложение двух версий чертежа с цветовым выделением изменений. Отличная функция для координации между дисциплинами.
3. **70%+ функций разработано на основе обратной связи от клиентов** — Fieldwire активно использует клиентский фидбэк для формирования дорожной карты.
4. **Полная работа в offline** — один из лучших offline-режимов среди конкурентов: просмотр планов, создание задач, маркировка — всё работает без интернета.
5. **Поддержка Hilti** — доступ к 120,000+ корпоративных клиентов Hilti для кросс-продаж.

### 1.5 UX/UI

**Общее впечатление:** Современный, чистый, интуитивно понятный интерфейс. Один из лучших в индустрии для полевых операций.

**Что сделано особенно хорошо:**
- Просмотр и маркировка чертежей — быстрый, плавный, без задержек
- Мобильное приложение — один из лучших мобильных опытов в индустрии
- Создание задач — буквально несколько кликов
- Навигация — логичная, не перегруженная
- Быстрый онбординг — новые пользователи начинают работать в течение 15-20 минут

**Что критикуют:**
- Инструмент рисования (pencil/drawing tool) — «extremely choppy», нет плавности для рукописных заметок
- Ограниченная кастомизация форм и отчётов
- Нельзя сортировать страницы планов — неудобно для крупных проектов
- Автоматическое именование листов иногда работает с ошибками

### 1.8 Слабые стороны и уязвимости

| Слабость | Детали |
|----------|--------|
| **Нет финансового модуля** | Отсутствуют биллинг, инвойсинг, бухгалтерский учёт, job costing |
| **Нет трекинга рабочего времени** | Встроенный time tracker отсутствует |
| **Нет зависимостей задач** | Невозможно задать порядок выполнения задач (task dependencies) |
| **Дорого для малых команд** | $89-94/user/мес — высокая цена по сравнению с Jobber, Connecteam |
| **API только для Enterprise** | Интеграция через API требует платного add-on и инженерной работы |
| **Ограниченная отчётность** | Нельзя создать отчёт по всем проектам одновременно |
| **Слабый ERP-модуль** | Нет закупок, складского учёта, HR — только полевые операции |
| **Фото из галереи** | «Cannot use a photo from their phone's storage and upload immediately» |

**Вывод по Fieldwire:** Отличный инструмент для полевых операций и управления задачами на стройплощадке, но не ERP-система. Уязвим перед комплексными решениями типа PRIVOD, которые покрывают финансы, HR, склад, закупки, планирование и аналитику в единой платформе.

- Использование оборудования
- Реальное время — синхронизация поле-офис

**Аналитика и отчётность:**
- Мощная библиотека стандартных отчётов
- Drill-down запросы
- Настраиваемые отчёты (требуют SQL/report writer)

- Дашборды

### 4.3 Уникальные особенности

1. **Часть экосистемы Trimble** — доступ к GPS (Trimble Pulse), IoT-датчикам, Tekla (BIM), SketchUp, ProjectSight и др. Ни один другой ERP-конкурент не имеет такой широкой экосистемы оборудования и технологий.
2. **Multi-company Management** — возможность управлять несколькими юридическими лицами и регионами в одной системе. Критично для крупных подрядчиков.
3. **Глубокое управление оборудованием** — один из лучших модулей Equipment Management на рынке, с автоматическим расчётом точки замены оборудования.
4. **Service Management** — полноценный модуль управления сервисными контрактами, диспетчеризации и мобильных нарядов. Мало кто из конкурентов это предлагает.
5. **Экстремальная настраиваемость** — пользователи могут настроить workflows, отчёты, формы, дашборды под свои процессы.

## КЛЮЧЕВЫЕ ВЫВОДЫ ДЛЯ PRIVOD

### Возможности для дифференциации

1. **Единая платформа vs. фрагментация:**
   - Fieldwire и PlanRadar — только полевые операции, без финансов/HR/склада
   - Sage 300/Intacct — только финансы, без полевых операций
   - Vista — полный ERP, но из 5+ отдельных продуктов с разным UX
   - **PRIVOD** = единая платформа с 421 сущностью, 229 сервисами

2. **Современный UX vs. Legacy:**
   - Sage 300 CRE и Vista — устаревший UI 2000-х годов
   - PRIVOD — React 19 + Tailwind + Dark Mode + Mobile-first

3. **Российская локализация:**
   - Ни один из конкурентов не поддерживает российское законодательство (КС-2, КС-3, АОСР, ГОСТ и т.д.)
   - PRIVOD — полная поддержка российской документации и интеграция с 1С, СБИС

4. **Ценообразование:**
   - Fieldwire: $39-94/user/мес (без финансов)
   - PlanRadar: $35-179/user/мес (без финансов)
   - Sage Intacct: $12,000-35,000/год (без полевых операций)
   - Vista: $10,000+ лицензия + $2,000-10,000/мес (legacy UI)
   - **PRIVOD может предложить единое решение по конкурентной цене**

5. **Мобильность и Offline:**
   - Fieldwire и PlanRadar имеют отличные мобильные приложения с offline
   - Sage и Vista значительно отстают в мобильности
   - PRIVOD должен обеспечить PWA/нативное приложение с offline не хуже Fieldwire

6. **Поддержка:**
   - Vista и Sage — основные жалобы на качество поддержки
   - Fieldwire — хорошая поддержка (благодаря Hilti)
   - **PRIVOD может выиграть за счёт русскоязычной поддержки и быстрого онбординга**
